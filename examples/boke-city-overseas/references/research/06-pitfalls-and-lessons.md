# 06 出海踩坑与教训

## 踩坑一：筹码交易与赌博定性危机（最大教训）

### 事件经过
| 时间 | 事件 |
|------|------|
| 2023年8月 | 印尼通信部（Kominfo）下架Higgs Domino Island，同时清理42,622条在线赌博内容 |
| 2024年2月 | Higgs Domino重新在Google Play上架，恢复电子系统运营商注册 |
| 2024年8月 | Kominfo再次封锁Higgs Domino Island |
| 持续 | 印尼警方逮捕23名涉嫌利用Higgs Domino进行在线赌博的嫌疑人，涉案交易额达800亿印尼盾（约500万美元，自2022年起） |

### 核心争议
- **官方立场**：B.I.G Technology Limited（Higgs Games开发商实体）通过律师声明，否认游戏属于赌博，强调已遵守印尼所有法规
- **第三方修改版**：开发商声称用于筹码交易的是第三方修改版本（非官方版），官方版本不提供用户间筹码转让功能
- **Change.org请愿**：印尼民众在Change.org发起请愿，要求关闭Higgs Domino并逮捕开发者

### 教训
1. **虚拟经济设计是合规生命线**：筹码系统必须从设计层面杜绝交易可能
2. **不能只控制App内**：即使App内无交易功能，第三方平台/改版也可能带来风险
3. **需要法律前置**：在印尼这样赌博全面禁止的市场，每个功能都需要法律合规审查
4. **品牌重建成本高**：从"Higgs Domino Island"更名为"Higgs Games Island"，用户认知需要重建

来源：CNBC Indonesia、Kominfo Aptika官方公告、Media Indonesia、Dagangberita

## 踩坑二：文化冲突与内容审查

### 印尼（穆斯林为主87%）
| 风险点 | 具体表现 | 应对 |
|--------|---------|------|
| 女性形象 | 露骨/暴露的女性角色在穆斯林文化中敏感 | 角色设计需要保守得体 |
| 赌博隐喻 | 任何与"赌"相关的视觉/文字都可能触发审查 | 去除所有赌博化表达 |
| 宗教敏感 | 不能出现猪/酒精/宗教冒犯内容 | 美术和文案全面审查 |
| 斋月期间 | 白天游戏行为可能被保守群体批评 | 斋月营销需要巧妙处理 |
| 节日尊重 | 开斋节等活动需要真诚尊重而非商业化利用 | 本地团队主导节日内容策划 |

### 菲律宾（天主教为主80%）
| 风险点 | 具体表现 | 应对 |
|--------|---------|------|
| 家庭价值观 | 菲律宾重视家庭，游戏不应鼓励沉迷 | 设计健康游戏提醒 |
| 宗教节日 | 圣周（Holy Week）期间需要敏感处理 | 减少促销活动，增加公益内容 |
| 政治敏感 | 菲律宾政治极化，避免任何政治立场 | 内容完全去政治化 |
| 反华情绪 | 部分存在（南海问题等） | 弱化中国开发商身份，强调本地化 |

来源：Morketing Global出海报告、行业分析

## 踩坑四：本地团队建设

### 常见问题
1. **管理风格冲突**：中国式加班文化在印尼/菲律宾不适用
   - 印尼劳动法保护性强，加班需要额外补偿
   - 菲律宾员工重视work-life balance
2. **沟通效率**：语言障碍导致需求传达偏差
   - 印尼团队需要印尼语沟通
   - 菲律宾团队英语能力强，沟通门槛低于印尼
3. **人才密度**：当地游戏开发人才储备有限
   - 核心技术岗位可能需要从国内外派
   - 运营/市场/客服必须100%本地化
4. **文化理解**：中国总部很难真正理解"mabar"文化
   - 需要信任本地团队的判断
   - 产品决策权需要适度下放

### 推荐架构
| 职能 | 建议配置 |
|------|---------|
| 产品决策 | 总部主导 + 本地输入 |
| 运营执行 | 100%本地化 |
| 美术设计 | 总部框架 + 本地审美审核 |
| 客服 | 100%本地语言 |
| 市场/UA | 本地团队主导 |
| 合规/法务 | 本地律所合作 |
| 技术开发 | 总部为主，本地辅助 |
| 数据分析 | 总部为主 |

来源：证券时报出海系列报道、GameRes行业分析

## 踩坑六：技术与基础设施

### 印尼技术挑战
| 挑战 | 说明 | 应对 |
|------|------|------|
| 网络质量 | 17,508个岛屿，偏远地区网速慢 | 优化包体大小+弱网适配 |

## 核心教训总结

### 血泪教训TOP 5
1. **合规是0或1的问题**：Higgs Domino被下架两次，每次都是致命打击——不是"做得够好"的问题，是"有没有做"的问题
2. **虚拟经济必须从设计层面合规**：不能靠"TOS禁止"来防止筹码交易，必须在技术层面让交易不可能发生
3. **本地化不是翻译**：选Domino而不是麻将，选Gaple而不是斗地主——这才是真正的本地化
4. **宗教文化是硬约束**：印尼穆斯林文化不是"尽量尊重"，是"必须遵守"，违反成本极高
5. **先做减法再做加法**：Higgs Domino一开始只做Domino品类，做到极致后才扩品类——不是一上来就做全品类平台

### 成功要素TOP 5
1. **选对赛道**：印尼Domino是蓝海（当时），而不是去印尼做MOBA/SLG红海
2. **平台聚合力**：一个App集成多种本地玩法，满足用户一站式需求
3. **社交驱动增长**：利用"mabar"文化，让社交关系成为留存和获客的引擎
4. **极致本地化**：美术/语言/支付/节日/运营全方位印尼化
5. **高频迭代**：保持版本更新节奏，持续响应用户反馈
