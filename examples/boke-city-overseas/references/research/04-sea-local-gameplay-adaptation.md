# 04 东南亚本地玩法适配

## 印尼国民级棋牌玩法全景

印尼有着深厚的棋牌文化传统，主要分为Domino类和扑克牌类两大体系。

来源：Pagat.com印尼国家/地区游戏页面

### Domino类游戏

#### Gaple（印尼Block Domino）

| 维度 | 规则 |
|------|------|
| 牌具 | 标准双六Domino牌组（28张） |
| 人数 | 4-5人（最常见4人） |
| 发牌 | 4人：每人6张（剩4张盖牌）或每人7张（无剩牌） |
| 5人：每人5张（剩3张盖牌） | |
| 起手 | 7张制：持有[0-0]者先出；6张制：翻开中央盖牌作为起始 |
| 玩法 | 单线连接，轮流在线两端接牌，接触端点数必须相同 |
| 无牌可出 | 弃一张牌面朝下（"死牌"，计分用），或直接Pass（变体） |
| 结束条件 | 某人出完所有牌 或 所有人都无法出牌 |
| 计分 | 未出牌的点数之和计分，[0-0]若是唯一空点牌则计25分 |
| 获胜 | 累计点数最低者获胜 |
| 变体 | 团队制（对家合作）/单人制/筹码结算制 |

来源：Pagat.com Gaple规则页

#### QiuQiu / KiuKiu（Domino 99）

| 维度 | 规则 |
|------|------|
| 牌具 | 28张印尼式Domino牌（厚卡纸印刷，约5.5cm x 2.9cm） |
| 人数 | 2-6人（6人最佳） |
| 关系 | 广东牌九（Pai Gow）的近亲玩法 |
| 起始 | 所有人下注（ante） |
| 发牌 | 先发3张面朝下 |
| 第一轮下注 | 看牌后可Check/Bet/Fold/Call/Raise（扑克式下注） |
| 补牌 | 未弃牌者再获发第4张牌 |
| 第二轮下注 | 同上 |
| **牌值计算** | 将4张牌分成2组各2张，每组总点数取个位数 |
| 示例 | [3-2]+[4-0] 总点数=3+2+4+0=9，牌值=9 |
| 最佳普通牌 | 两组均为9（即9,9） |
| 获胜 | 最高牌值者赢得底池；平局则平分 |

**特殊牌型（从高到低）**：
1. **Enam Dewa（六神）**：[3-3],[4-2],[5-1],[6-0] — 四张牌每张两端之和均为6
2. **Empat Balak（四对子）**：四张均为双数牌（如[1-1],[2-2],[3-3],[4-4]）
3. **Murni Besar（纯大）**：四张牌总点数>=39
4. **Murni Kecil（纯小）**：四张牌总点数<=9

来源：Pagat.com QiuQiu规则页、Gamerules.com、维基百科Kiu kiu词条

### 扑克牌类游戏

#### Capsa Susun（印尼十三张/Chinese Poker）

| 维度 | 规则 |
|------|------|
| 牌具 | 标准52张扑克牌 |
| 人数 | 2-4人 |
| 发牌 | 每人13张 |
| 摆牌规则 | 分成3墩：底墩5张 > 中墩5张 > 头墩3张 |
| 牌力要求 | 底墩必须强于中墩，中墩必须强于头墩 |
| 犯规 | 违反大小顺序 → 该轮自动输 |
| 牌型 | 标准扑克牌型：高牌 < 一对 < 两对 < 三条 < 顺子 < 同花 < 葫芦 < 四条 < 同花顺 |
| 特殊牌型 | 三同花/三顺子/全高/全低/六对/五对一三条/全黑/全红/多个四条 |
| 比较 | 逐墩比较，赢多墩者获胜 |

来源：Pagat.com、Google Play Capsa Susun游戏描述、维基百科Chinese Poker

## 中国玩法 vs 印尼玩法的设计差异

| 维度 | 中国棋牌 | 印尼棋牌 |
|------|---------|---------|
| 核心牌具 | 麻将+扑克牌 | Domino牌+扑克牌 |
| 国民级品类 | 斗地主/麻将 | Gaple/QiuQiu |
| 数值系统 | 麻将番数计算（指数级） | Domino点数计算（线性） |
| 下注节奏 | 快速（一把几十秒到几分钟） | 中等（QiuQiu有两轮下注） |
| 社交属性 | 麻将=4人固定社交 | Domino=4-5人弹性社交 |
| 运气vs技巧 | 麻将偏技巧，斗地主平衡 | Gaple偏策略，QiuQiu偏运气 |
| 单局时长 | 斗地主3-5分钟/麻将5-15分钟 | Gaple5-10分钟/QiuQiu2-5分钟 |
| 观战价值 | 麻将高（复杂策略） | QiuQiu低/Capsa Susun中高 |

## 玩法数值化设计要点

### Gaple数值设计
- 28张牌组合固定，策略空间在于记牌和出牌顺序
- [0-0]的25分惩罚规则创造了独特的博弈张力
- 团队制增加了沟通/配合维度
- **设计启发**：简单规则+深度博弈，适合短session反复游玩

### QiuQiu数值设计
- 个位数取值（0-9）创造了9,9的天花板感
- 特殊牌型设计（六神/四对子/纯大纯小）增加了Jackpot兴奋感
- 两轮下注+3→4张补牌创造了决策点
- **设计启发**：高方差+低复杂度=高付费转化（赌性强但不需要高技巧）

### Capsa Susun数值设计
- 13张分3墩的组合空间极大（策略深度高）
- 大小递减约束（底>中>头）创造了取舍张力
- 特殊牌型bonuses激励追求极端组合
- **设计启发**：高技巧天花板，适合核心玩家和竞技化
