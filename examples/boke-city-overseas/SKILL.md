## 二、印尼出海路径拆解

### 为什么是印尼 + 为什么是Domino

**市场面**：
- 2.74亿人口 × 56%游戏渗透率 = 1.55亿玩家
- 占东南亚游戏市场45.8%
- Mobile-only（不是Mobile-first）
- 信用卡渗透率极低 → 电子钱包是唯一通道

**文化面**：
- Domino（特别是Gaple和QiuQiu）是印尼国民级休闲活动
- "Mabar"（一起玩）是社交仪式，不是打发时间
- 斋月期间游戏活跃度峰值（最佳获客窗口）

**竞争面**（2018年入场时）：
- Domino数字化品类没有绝对头部
- Boyaa等先行者产品力/运营力不足
- 蓝海窗口期明确

### 本地化四层模型

```
┌─────────────────────────────────────────┐
│  第4层：运营节奏                          │
│  开斋节活动 / 独立日赛事 / 斋月专题       │
├─────────────────────────────────────────┤
│  第3层：支付本地化                        │
│  GoPay(88%) > DANA(83%) > OVO            │
│  + 便利店代收 + 运营商代扣                │
├─────────────────────────────────────────┤
│  第2层：美术/语言                         │
│  热带岛屿主题 / 全印尼语UI / 本地审美     │
├─────────────────────────────────────────┤
│  第1层：玩法核心                          │
│  Domino Gaple + QiuQiu + Capsa Susun     │
│  （不是斗地主/麻将的翻译版）              │
└─────────────────────────────────────────┘
```

**关键认知**：本地化不是第2层（翻译），是第1层（玩法选择）。波克没有把斗地主搬到印尼，而是为印尼做了Domino。

## 四、东南亚本地玩法速查

### 印尼三大玩法

**Gaple（Block Domino）**
- 4-5人，双六Domino牌28张
- 单线连接出牌，匹配端点数
- 策略核心：记牌 + 出牌顺序 + [0-0]的25分惩罚博弈
- 设计DNA：简单规则，深度博弈，短session反复玩

**QiuQiu / KiuKiu（Domino 99）**
- 2-6人，28张Domino牌，牌九近亲
- 4张牌分2组，取个位数，9,9最大
- 特殊牌型：六神 > 四对子 > 纯大(>=39) > 纯小(<=9)
- 两轮下注（3张→4张），扑克式下注
- 设计DNA：高方差 + 低复杂度 = 高付费转化

**Capsa Susun（十三张/Chinese Poker）**
- 2-4人，52张扑克牌
- 13张分3墩：底5张 > 中5张 > 头3张
- 违反大小顺序 → 犯规自动输
- 设计DNA：高技巧天花板，适合核心/竞技玩家

### 菲律宾三大玩法
- **Pusoy Dos**：大老二变体，4人对战
- **Tong-its**：菲律宾特色Rummy变体
- **Pusoy**：十三张（与Capsa Susun基本相同）

## 启发式规则

### 出海决策启发式

1. **"Domino不是斗地主"法则**：进入新市场的第一件事不是翻译你的产品，而是找到当地的"斗地主"是什么。波克选Domino而非推斗地主出海，这个决策价值超过所有后续运营。

2. **"一个App多个玩法"法则**：延续平台大厅思路，一个App聚合3-5种本地玩法。用户只装一个App就能满足所有棋牌需求，这是留存和竞争壁垒的核心。

3. **"Ramadan即春节"法则**：每个市场都有自己的"春节"——最高活跃期。印尼是斋月Ramadan，菲律宾是圣诞节。把最大的运营资源投入这个窗口。

4. **"58美元 vs 140美元"法则**：不同市场的人均游戏支出差异巨大。印尼做广告为主($58/年)，菲律宾可以做内购为主($140.7/年)。变现模型必须跟着市场走。

5. **"筹码不可交易是生命线"法则**：Higgs Domino两次被下架的教训——虚拟经济设计必须从技术层面杜绝筹码交易，不是靠TOS禁止。这是0或1的合规问题。

6. **"GoPay优先于PayPal"法则**：东南亚支付不是信用卡也不是PayPal，是本地电子钱包。印尼接GoPay(88%)/DANA(83%)，菲律宾接GCash(8100万MAU)。不接这些等于不要钱。

7. **"弱化中国身份"法则**：用本地化品牌（Higgs Games而非波克城市），用本地公司实体（B.I.G Technology），让用户感觉这是"我们的App"而不是"中国App"。在存在反华情绪的市场尤其重要。

8. **"本地团队决定本地运营"法则**：运营/市场/客服必须100%本地化，产品决策权需要适度下放。中国总部再优秀，也理解不了"mabar"文化的深层含义。

9. **"低端机是基本盘"法则**：印尼/菲律宾大量用户使用低端Android设备。安装包<100MB、适配Android 4.4+、弱网优化——这些技术细节直接决定可触达用户量。

10. **"先做一个市场做透"法则**：波克先在印尼做到Domino品类第一，再考虑扩展。不要同时进多个国家。每个东南亚国家都是独立市场，语言/文化/支付/合规全不同。
