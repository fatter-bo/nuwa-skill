# 研究维度三：裂变机制设计

| 指标 | 数据 | 说明 |
|------|------|------|
| 普通小游戏社交获客占比 | 30%-50% | 新用户来自社交渠道 |
| 棋牌类社交获客占比 | 70%-80% | 社交属性天然更强 |

- **优化后产品**：可大幅超越基准值

## 2. 核心裂变机制类型

### 2.1 竞技排行类

**好友排行榜**
- 最基础也最有效的社交机制
- 利用"超越好友"心理驱动自然传播
- 跳一跳的DAU 1.7亿很大程度依赖排行榜
- 需要调用微信关系链数据

**地区/战队排行**
- 羊了个羊的地区排名激发地域归属感
- 战队对抗创造集体荣誉感
- 扩大了竞争维度，从个人到群体

### 2.2 互助协作类

**擂台赛（异步+实时融合）**
- 裂变效率比传统好友房高出5倍
- 更符合微信异步交流的特性
- 用户无需同时在线即可参与
- 降低了社交互动的门槛

**好友房PK**
- 实时对战模式
- 社交体验最强但用户撮合难度高
- 适合核心社交场景

### 2.3 分享获益类（需注意红线）

**合规的分享设计**：
- 分享后获得非必要性奖励（不影响核心游戏进程）
- 分享展示成就（排名、成绩截图）
- 分享邀请好友组队（功能性邀请）

**违规的分享设计**（已被微信禁止）：
- 分享复活（强制分享才能继续游戏）-- 已被取消回调
- 分享解锁关卡（强制分享才能进入下一关）
- 分享获得大量金币/道具（利诱分享）

## 3. 分享卡片设计策略

### 动态分享卡片

微信支持游戏在分享后持续更新群聊中的卡片内容，显示实时进度，鼓励持续参与。

### 多样化分享格式

| 分享形式 | 用途 | 效果 |
|---------|------|------|
| 成绩海报 | 炫耀成就 | 社交货币效应 |
| 表情包/动态贴纸 | 嵌入日常对话 | 长期品牌曝光 |
| 挑战邀请卡 | 邀请好友PK | 直接导流 |
| 进度展示卡 | 展示游戏进度 | 好奇心驱动 |

### 分享率优化要点

1. **降低分享摩擦**：一键分享，减少操作步骤
2. **提供分享动机**：分享的内容本身有社交价值（有趣/有用/值得炫耀）
3. **选择分享时机**：在情绪峰值时触发分享提示（通关、打破纪录、获得稀有物品）
4. **差异化分享内容**：分享给不同好友/群的卡片内容可以不同

**来源**：腾讯游戏学堂 - 裂变方法论；微信官方分享
**可信度**：高 -- 腾讯官方分享的最佳实践

## 4. 诱导分享的限制与演变

### 历史演变

```
2018年初：分享复活机制泛滥 → 小游戏"群毒瘤"现象
    ↓
2018年中：微信取消分享回调数据 → 无法判断用户是否真的分享
    ↓
2019-2020年：分享机制转向"价值分享"而非"强制分享"
    ↓
2021-2025年：平台推出社交组件（群PK、群任务、送礼等），
             引导开发者用官方工具实现社交传播
```

### 当前合规边界

**可以做的**：
- 展示好友排行榜
- 提供"邀请好友"入口
- 分享成绩/成就截图
- 群内异步PK/擂台赛
- 好友间送礼/互助（使用官方组件）

**不能做的**：
- 强制分享才能继续游戏
- 分享获得游戏内利益
- 弹窗大图诱导分享
- 利用红包/金钱诱导分享
- 伪造"分享成功"提示

**来源**：微信外部链接内容管理规范；微信小游戏平台规则
**可信度**：高 -- 微信官方规范文档

| 朋友圈分享 | 分享到朋友圈 | 广域曝光 |
| 搜索发现 | 搜索推荐优化 | 自然流量 |
