# 05 各国合规边界

## 核心命题

通胀型、不可交易的链上积分在法律上如何定性？它既不是加密货币（不可交易），也不完全是传统游戏积分（数据在链上）。各国监管态度差异巨大。

---

## 法律定性分析

### 通胀型不可交易积分的独特属性

| 属性 | 加密货币 | 传统游戏积分 | 链上欢乐豆 |
|------|---------|------------|-----------|
| 可交易 | 是 | 一般否（但灰色地带） | 合约级禁止 |
| 有限供给 | 通常是 | 无限 | 无限 |
| 可兑法币 | 是 | 不可 | 不可 |
| 链上记录 | 是 | 否 | 是 |
| 投资属性 | 有 | 无 | 无 |
| 用户购买 | 交易所 | 可能（内购） | 可能（内购） |

**关键论点**：链上欢乐豆更接近"传统游戏积分的链上记账版"，而非"加密资产"。

## 各国/地区合规地图

### 绿色区域 — 可直接运营

#### 美国
- **定性**：不可交易的游戏积分不被SEC视为证券（不通过Howey Test）
- **关键条件**：不可交易 + 无投资回报预期 + 无二级市场
- **FTC要求**：内购价格透明，虚拟货币显示真实价格
- **各州差异**：部分州（NY, CA）可能要求额外披露
- **风险等级**：低

#### 日本
- **定性**：非"虚拟资产"（不可交易排除了资金决济法适用）
- **游戏相关**：受"景表法"约束，限制gacha/抽奖机制
- **关键条件**：积分不可变现 + 明确标注"游戏内专用"
- **风险等级**：低

#### 新加坡
- **定性**：MAS不将不可交易积分视为"数字支付代币"
- **条件**：不具支付功能 + 不可兑换法币
- **风险等级**：低

#### 巴西
- **定性**：游戏积分不受证券监管
- **条件**：不涉及投资合同
- **风险等级**：低

### 黄色区域 — 需注意合规细节

#### 欧盟 (MiCA + CPC Guidelines)
- **MiCA**：2024年12月全面生效，但主要针对可交易加密资产
- **CPC 7原则（2025）**：
  1. 虚拟货币内购必须显示真实货币价格
  2. 不得使用误导性定价
  3. 允许用户购买精确数量（不强制超额购买）
  4. 购买前充分告知
  5. 条款用清晰语言书写
  6. 14天退款权
  7. 保护未成年人
- **Digital Fairness Act（预计2026）**：可能进一步收紧
- **关键**：即使积分不可交易，用户用法币购买积分的环节受消费者保护法约束
- **风险等级**：中（合规成本较高但可执行）

#### 韩国
- **虚拟资产用户保护法（2024）**：主要针对可交易虚拟资产
- **游戏法规**：GRAC评级、消费上限（18岁以下月7万韩元，成人月50万韩元）
- **灰色地带**：链上数据是否使积分被视为"虚拟资产"尚无明确判例
- **风险等级**：中

#### 印度
- **加密税**：30%的加密收益税（2022 Finance Act），但不可交易积分是否适用存疑
- **无明确游戏虚拟货币法规**
- **风险等级**：中（税务定性不确定）

### 红色区域 — 高风险或不可操作

#### 中国大陆
- **全面禁止加密货币交易和挖矿**（2021年）
- **游戏版号制度**：所有游戏需获得版号
- **虚拟货币管理**：不得提供虚拟货币的发行、交易服务
- **关键问题**：链上积分即使不可交易，"使用区块链技术"本身可能触发监管审查
- **建议**：中国大陆版本使用纯链下方案，海外版本上链
- **风险等级**：高

#### 阿联酋
- **VARA监管框架（2023）**：所有虚拟资产活动需VARA许可
- **不确定**：不可交易积分是否属于"虚拟资产"定义范围
- **风险等级**：高（许可成本高且不确定）

## 合规设计框架

### "合规优先"架构

```
全球统一：
- 积分合约级不可交易（_beforeTokenTransfer限制）
- 不提供法币兑换出口
- 不接入DEX或任何交易协议
- 合约代码开源（证明不可交易性）

按区域调整：
- 内购定价页面：显示当地法币价格（满足EU CPC原则1）
- 消费上限：韩国用户适用GRAC上限
- 未成年人保护：全球启用家长控制
- 退款政策：EU用户14天退款权
- 数据存储：遵守GDPR（EU）/ PIPL（中国）
```

### 防止积分被定性为"证券"的三条红线

1. **绝不暗示投资回报**：积分不升值，不分红，不与收入挂钩
2. **绝不开放二级市场**：合约级禁止P2P转账，不上任何DEX
3. **绝不承诺回购**：积分不可兑回法币，无"退币"机制
