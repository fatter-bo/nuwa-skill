> 增长的本质不是"花更多钱买更多用户"，而是在CPI、留存和ARPDAU三个杠杆之间找到最优平衡点。同样$10万预算，投在错误的渠道/市场/创意上，与投在正确的组合上，ROAS可以差5-10倍。

## 一、付费买量基础

### 1.1 广告平台选型矩阵

| 平台 | iOS CPI | Android CPI | 核心优势 | 最佳场景 |
|------|---------|-------------|---------|---------|
| Meta (FB/IG) | $2.00-5.50 | $1.00-2.50 | 受众规模最大、定向精准 | 欧美核心获客 |
| Google Ads | $1.50-4.50 | $0.80-2.00 | 覆盖渠道最广（搜索+YouTube+Play） | 全渠道覆盖 |
| TikTok | $0.50-2.00 | $0.20-1.00 | 年轻用户、UGC适配、极低CPI潜力 | 超休闲大规模获客 |
| AppLovin | $1.00-3.00 | $0.50-1.50 | AXON AI优化、Playable广告59%流量 | 游戏垂直深耕 |
| Unity Ads | 中等 | 中等 | 游戏引擎原生集成 | 开发者生态 |

### 1.2 CPI趋势与基准（2025-2026）

- 2025年全球游戏CPI升至$0.56，同比上涨30%
- 移动游戏行业UA总支出达$250亿
- 付费/自然安装比从2.07跃升至3.33，增长61%
- iOS CPI约为Android的5-8倍（ATT影响）

**按品类CPI**：

| 品类 | 平均CPI |
|------|---------|
| 超休闲 | $0.15-0.50 |
| 休闲 | $0.50-2.00 |
| 中核休闲 | $1.50-4.00 |

### 1.3 广告素材策略

**推荐素材配比**：50%视频 + 30%Playable/Interactive + 20%静态图

**视频广告结构公式**：Hook(3s) -> 玩法展示(10-15s) -> 情感高潮/失败重试(5s) -> CTA(3s)

**Playable广告**：
- D7 ROAS约17%（休闲益智类），优于纯视频
- 创意寿命平均15.4天，远优于UGC的7.6天
- 30秒内可体验核心玩法、操作极简

**创意产出节奏**：
- 日消耗>$10K：每月至少15-20个视频变体 + 3-5个Playable变体
- 78%保持顶级表现的Campaign每周刷新创意
- 低于10个月新变体的账户，60天内CPI上升22%

### 1.4 ROAS优化

**ROAS三大驱动因素**：

| 因素 | 影响 | 优化杠杆 |
|------|------|---------|
| CPI | 分母 | 创意优化、受众精准、平台选择 |
| 留存 | 间接影响分子 | 产品优化、推送策略、LiveOps |
| ARPDAU | 直接影响分子 | 变现策略、广告频次、IAP定价 |

**优化方法**：分渠道ROAS追踪 -> 分创意ROAS分析 -> 混合变现ROAS（IAP+广告收入合并） -> ML预测D30+ ROAS

## 二、ASO优化

### 2.1 双平台算法差异

| 维度 | App Store (iOS) | Google Play |
|------|----------------|-------------|
| 标题长度 | 30字符 | 30字符 |
| 副标题/短描述 | 30字符Subtitle | 80字符Short Description |
| 关键词字段 | 100字符隐藏字段 | 无（靠Description） |
| Description索引 | 不索引 | 索引 |
| 算法权重 | 下载量+评分+关键词相关性 | 留存+稳定性+更新频率+关键词 |

**2025-2026关键变化**：Apple为CPP（Custom Product Pages）引入关键词链接，CPP出现在自然搜索结果中，从付费工具变为ASO核心资产。

### 2.2 关键词策略

- 2026年ASO重心从短词转向长尾查询，堆砌已失效甚至反向惩罚
- **标题公式**：品牌名 + 核心关键词（如"Merge Town: 合成消除游戏"）
- **更新频率**：每2-4周基于数据调整

**关键词研究流程**：种子词生成 -> 竞品TOP10拆解 -> 搜索量vs竞争度矩阵 -> 本地化变体 -> 季节性关键词

### 2.3 截图与视频

- 前两张截图是转化率决定性因素
- 游戏截图需展示核心玩法（非菜单/Loading画面）
- 57%的游戏在Google Play进行截图A/B测试，头部游戏高达8次/年

**A/B测试优先级**：图标（P0，15-25%提升潜力）-> 前两张截图（P0，10-20%）-> 视频预览（P1）-> 副标题（P1，5-10%）

### 2.4 评分管理

- 评分<3.5星：搜索可见性显著降低
- 评分>4.5星：进入"编辑推荐"候选池的基本门槛
- **弹窗时机**：完成成就后、连续登录3天+的活跃玩家；**避免**：首次打开、付费失败后、卡关时
- 24小时内回复所有1-2星评论，修复后请求更新评分

## 三、社交裂变设计

### 3.1 K-Factor基础

```
K = i(邀请率) x c(转化率)
```

- K < 1：需持续买量维持；K = 1：自我维持；K > 1：指数增长
- Cycle Time（裂变周期）越短，增长速度越快

**K-Factor基准**：

| 游戏类型 | 典型K-Factor | 典型Cycle Time |
|---------|-------------|---------------|
| 超休闲 | 0.1-0.3 | 1-2天 |
| 休闲 | 0.05-0.2 | 3-7天 |
| 社交休闲 | 0.3-0.8 | 1-3天 |
| 微信小游戏（裂变强） | 0.5-2.0+ | 0.5-2天 |

### 3.2 各平台裂变方式

| 平台 | 核心裂变机制 | 覆盖市场 |
|------|------------|---------|
| 微信 | 群分享、好友助力、排行榜、红包裂变、小程序卡片 | 中国 |
| Facebook | Instant Games、好友挑战、成就分享、邀请奖励 | 欧美/东南亚/拉美 |
| LINE | 官方账号、游戏角色贴纸、好友邀请、LINE Points | 日本/泰国/台湾/印尼 |
| KakaoTalk | 私人消息邀请、排行榜、心/能量互赠 | 韩国（74%渗透率） |

### 3.3 四种裂变设计模式

1. **互惠裂变**：双方获得奖励，降低"打扰好友"心理障碍。奖励可递增（前5个好友递增）
2. **助力裂变**：需好友帮助完成动作（复活/解锁），助力动作尽量轻量（点击即可）
3. **竞争裂变**：排行榜限制好友圈（150人内），每周重置保持新鲜感
4. **内容裂变**：生成个性化内容（战绩卡/角色形象），附带deeplink一键入口

### 3.4 K-Factor优化

**提升i（邀请率）**：一键分享、成就时机弹出、社交货币设计、多触点嵌入

**提升c（转化率）**：深度链接直达、社交证明（"XX好友在玩"）、Instant Game无需下载、首次体验优化

**缩短Cycle Time**：新手引导嵌入分享、早期社交绑定、每日分享激励

## 四、交叉推广

### 4.1 核心价值

零成本获客 + 高预测性 + 用户质量优于外部流量 + Portfolio LTV最大化。前提：工作室拥有2款以上活跃产品。

### 4.2 广告位设计与触发时机

**高转化时机**：
1. 关卡完成/里程碑达成后（正向情绪+自然断点）
2. 能量耗尽时（等待恢复期的替代选择）
3. 连续失败后（"换个口味"）
4. 会话结束时（退出前最后触点）

**核心原则**：在玩家即将离开游戏之前展示交叉推广，推另一款游戏而非让玩家流失到竞品。

### 4.3 用户分层推荐

| 用户阶段 | 推荐策略 |
|---------|---------|
| 新用户（D1-D7） | 不推荐（保护留存） |
| 活跃用户（D7-D30） | 轻度曝光、品牌认知 |
| 成熟用户（D30+） | 主要推广对象 |
| 流失预警用户 | 积极推广替代产品 |

**频控**：单日展示上限2-3次，同一游戏间隔>=24小时，负反馈关闭后7天不再展示

### 4.4 Portfolio LTV思维

```
Portfolio LTV = 游戏A的LTV + 交叉推广转化概率 x 游戏B的LTV + ...
```

即使单款游戏LTV不足以覆盖CPI，通过交叉推广延长用户在组合内的生命周期，整体可能盈利。

## 五、留存与LTV

### 5.1 留存基准（2025-2026）

| 品类 | D1（目标） | D7（目标） | D30（目标） |
|------|-----------|-----------|------------|
| 超休闲 | 20-30% | 3-5% | <2% |
| 休闲（Puzzle/Merge） | 35-45% | 12-18% | 6-10% |
| 中核休闲 | 30-40% | 10-15% | 5-8% |

行业中位数：D1约22%，D7约3.4-3.9%。TOP10%：D1约40%，D7约12%。

### 5.2 留存优化杠杆

**D1（首次体验）**：30秒内完成核心操作并获得正反馈（+5-15%）、渐进式教学（+3-8%）、减少加载/崩溃（+3-10%）

**D7（核心循环深度）**：确保7天内不消耗完内容（+3-8%）、社交绑定（+2-5%）、递增签到奖励（+2-4%）、限时活动（+2-5%）

**D30（长期驱动）**：Meta层深度（叙事/建造/收集）（+2-5%）、Battle Pass（+3-7%）、每周新内容/活动（+2-5%）

**回流礼包梯度**：1-3天离线轻度 -> 3-7天中度 -> 7-14天重度 -> 14天+重新引导

### 5.3 LTV预测方法

**方法一：系数法** — K = D90 LTV / D3 LTV，新群组D3 LTV x K = 预测D90 LTV

**方法二：留存曲线拟合** — 用幂函数拟合D1-D7留存，预测留存 x ARPDAU求和

**方法三：机器学习** — D0-D3行为特征训练模型，精度可达95%

**品类LTV参考（美国市场）**：

| 品类 | 典型D180 LTV |
|------|-------------|
| 超休闲 | $0.50-2.00 |
| 休闲 | $3.00-15.00 |
| 中核休闲 | $8.00-25.00 |
| 合成游戏头部 | $15.00-40.00 |

### 5.4 盈利决策框架

```
利润 = LTV - CPI - 运营成本
D_n ROAS = D_n LTV / CPI x 100%
```

| 信号 | 行动 |
|------|------|
| D7 ROAS > 目标 | 放量 |
| D7 ROAS < 目标但D30达标 | 保持并优化前端 |
| LTV/CPI > 1但利润薄 | 同时优化CPI和变现 |
| LTV/CPI < 1 | 暂停该渠道/创意 |

## 七、核心框架总览

### 框架一：ROAS三杠杆模型
ROAS = 收入/广告支出，由CPI（分母）、留存（间接分子）、ARPDAU（直接分子）三个杠杆驱动。优化顺序：先降CPI（创意+渠道），再提留存（产品），最后提ARPDAU（变现）。

### 框架二：UA渠道选型决策树
```
预算充足（>$50K/月）→ Meta+Google基础组合+AppLovin补充
预算有限（<$50K/月）→ TikTok低CPI获量+ASO自然流量
超休闲品类 → TikTok为主（CPI最低$0.20）
中核休闲 → Meta为主（定向精准+留存质量高）
iOS优先市场（日本） → Meta+AppLovin（iOS端37%份额）
Android优先市场（东南亚） → Google+TikTok
```

### 框架三：创意生命周期管理
创意产出 -> 小预算测试（$500-1000/创意） -> 识别高ROAS创意 -> 放大投放 -> 监控衰减（CTR下降30%=更换） -> 迭代新变体。Playable寿命15.4天，UGC寿命7.6天，视频居中。每月保持15-20个新变体。

### 框架四：K-Factor裂变优化飞轮
提升邀请率（降低分享摩擦+社交货币设计） -> 提升转化率（深度链接+社交证明+无下载体验） -> 缩短裂变周期（新手引导嵌入分享） -> K-Factor提升 -> 自然用户增长 -> 降低混合CPI -> 释放预算投入付费买量 -> 循环

### 框架五：留存分层优化体系
D1优化FTUE（30秒正反馈） -> D7优化核心循环（内容深度+社交绑定+每日激励） -> D30优化Meta层（叙事/收集/Battle Pass+LiveOps节奏） -> 流失预警回流机制（离线礼包梯度+推送策略）

### 框架六：全球市场分阶段进入模型
测试市场（低CPI英语区）-> 验证市场（美国LTV/ROAS验证）-> 规模化市场（西欧+日韩高LTV） -> 全球化市场（中东/东南亚/拉美本地化）。每阶段达标后再进入下一阶段，不同时打3+市场。

### 框架七：Portfolio LTV最大化模型
单款游戏LTV可能不足以覆盖CPI，但通过交叉推广延长用户在产品组合内的生命周期。Portfolio LTV = 各游戏LTV x 交叉转化概率之和。前提：2款+活跃产品、用户分层推荐（D30+成熟用户为主推对象）、频控保护源游戏留存。

## 八、决策启发式

### 1. "CPI是温度计，LTV是体检报告"
CPI反映市场竞争烈度的即时温度，LTV反映产品健康的根本状态。只看CPI会误入低质量用户陷阱，只看LTV会错失规模化窗口。两者必须联动看：D7 ROAS = D7 LTV / CPI是核心决策指标。

### 2. "创意衰减不等人"
78%保持顶级表现的Campaign每周刷新创意。低于10个月新变体的账户，60天内CPI上升22%。创意产出是一场军备竞赛，停下来就是倒退。

### 3. "Playable值两倍、活两倍"
Playable广告创意寿命15.4天（UGC仅7.6天），且D7 ROAS约17%优于纯视频。虽然制作成本高，但摊到生命周期的单位成本更低。预算有限时优先做Playable而非铺量UGC。

### 4. "ASO是免费的第二增长引擎"
付费/自然安装比从2.07跃升至3.33，意味着每买3.33个用户只有1个是自然来的。做好ASO（图标/截图/关键词/评分）可以显著提升自然流量占比，本质上是在降低混合CPI。

### 5. "裂变K值0.3就是赚到"
K-Factor不需要达到1才有价值。K=0.3意味着每买10个用户自动带来3个免费用户，等效CPI降低23%。任何能把K值从0.05提升到0.2的产品改动，都值得投入。

### 6. "D1留存40%是分水岭"
D1留存40%以上的休闲游戏才值得大规模买量。D1<30%时先优化FTUE（30秒正反馈），D1在30-40%间优化教学和首日目标，D1>40%再放心投放。在FTUE没做好之前买量就是烧钱。

### 7. "交叉推广只推成熟用户"
D1-D7新用户看到交叉推广会损害留存。D30+成熟用户品牌忠诚度已建立，是交叉推广的最佳对象。流失预警用户则是"最后一搏"——推另一款游戏总好过完全流失。

### 8. "斋月是中东的双十一"
斋月期间游戏时长大幅增加（日间休息+夜间活跃），开斋节是消费高峰。做中东市场的团队必须围绕斋月做全年最大的营销投入，错过这个窗口等于浪费大半年。

### 9. "先低CPI市场测试，再高LTV市场变现"
澳大利亚/加拿大/东南亚英语区的CPI低、用户行为接近欧美，是理想的测试市场。在低成本环境验证留存和变现模型后，再进入美国/日本等高LTV市场才有把握。反过来做是用最贵的流量做实验。

### 10. "混合变现ROAS才是真实ROAS"
休闲游戏70%+收入可能来自广告。只算IAP的ROAS会严重低估广告变现型用户价值，导致错误地暂停实际盈利的Campaign。必须IAP+广告收入合并计算ROAS。
