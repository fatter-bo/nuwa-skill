**用户说"我想做空XXX"时，按以下顺序执行：**

```

  → 每层给出 ✅通过 / ⚠️存疑 / ❌不通过
```

## 二、轧空风险评估（做市商反杀检查）

**开空之前必须确认——做市商有没有拉盘的动机和能力。**

### 6条做市商反杀检查

| # | 条件 | 检查方法 |
|---|------|---------|
| 1 | 做市商近期刚完成吸筹（链上库存增加） | Arkham追踪做市商钱包余额 |
| 2 | 资金费率已极负（空头拥挤，轧空利润空间大） | Coinglass FR数据 |
| 3 | 项目方近期有重大利好待发布 | WebSearch项目公告/社区 |
| 4 | 流动性集中在少数地址（做市商控盘度高） | Etherscan持仓分布 |
| 5 | 做市协议刚续约或换新做市商（有做业绩的动力） | 项目公告/链上推断 |
| 6 | 订单簿买盘异常厚但价格不涨（悄悄吸筹） | 订单簿深度分析 |

**判断标准**：
- 0-1条满足 = 轧空风险低（可以做空）
- 2-3条满足 = 轧空风险中高（极小仓位或放弃）
- 4+条满足 = **⛔ 禁止做空**

## 三、做空工具选择框架

### 决策树

```
你要做空多久？
├─ <7天 → 永续合约（流动性最好）
├─ 7-30天 → 永续合约（注意FR成本）或交割合约（无FR风险）
├─ >30天 → Put期权（成本确定）或滚动交割合约
└─ 小币/流动性差 → 必须用Put期权（避免被定点爆破）
```

### 杠杆倍数计算公式

```
最大杠杆 = 可承受亏损比例(%) / 止损百分比(%)

示例1: 愿意亏总资金2%，止损设5%
→ 最大杠杆 = 2/5 = 0.4x（不加杠杆，甚至要缩仓）

示例2: 愿意亏总资金3%，止损设10%
→ 最大杠杆 = 3/10 = 0.3x

示例3: 愿意亏总资金5%，止损设5%（高风险）
→ 最大杠杆 = 5/5 = 1x（刚好1倍，无杠杆）
```

**铁律**：币圈scam wick频繁，实际止损宽度应为技术止损位的2-3倍ATR。反算后大部分情况下你不应该加杠杆。

### 虚值Call对冲（做空保险）

做空时同时买入OTM看涨期权，为轧空风险设置硬上限：

```
示例：做空ETH $2000，买入$2200 Call（OTM 10%）
- Call成本：约$50/ETH（2.5%）
- 如果ETH暴涨到$3000：空头亏$1000，但Call盈利$800，净亏$200+$50=$250
- 没有Call的话：净亏$1000
- 对冲成本=保险费，值得
```

## 四、仓位管理铁律

| 规则 | 标准 | Jim Chanos参考 |
|------|------|---------------|
| 单笔做空仓位上限 | 总资金的2-5% | Chanos: 0.5-5% per position |
| 同方向最大敞口 | 总资金的15% | 多个空头仓位总和不超过15% |
| 分批建仓 | 30/30/40三批 | 第一批试探，确认后加仓 |
| 浮盈加仓条件 | 已盈利>5% + 原始止损已移到成本线 | 盈利时才加，不摊薄亏损仓位 |
| 止损触发后冷静期 | 最少24小时不得重新开同方向仓位 | 防止报复性做空 |

## 五、止损纪律

### 四种止损类型

| 类型 | 适用场景 | 规则 |
|------|---------|------|
| **硬止损** | 所有做空仓位的底线 | 开仓时就设好，不移动，不取消 |
| **移动止损** | 浮盈后保护利润 | 盈利>10%后，止损移至成本线；盈利>20%后，止损移至盈利10%处 |
| **时间止损** | 横盘消耗资金费率 | 持仓超过预设天数（如14天）且未达目标，减半或平仓 |
| **波动率止损** | 高波动代币 | 止损宽度=2-3倍ATR，而非固定百分比 |

### "我再等等"心理的对抗

- **预设规则自动执行**：用交易所条件单，不靠人工判断
- **写下决策理由**：开仓前写下"我为什么做空"和"什么情况下我认错"
- **开仓后不看利多消息**：不是因为信息不重要，而是因为你的确认偏误会过滤掉它

| 陷阱 | 表现 | 对抗方法 |
|------|------|---------|
| 均值回归偏见 | "涨这么多了一定会跌" | 趋势可以持续远超你的预期。看流动性方向，不看涨幅 |

## 诚实边界

- **做空本质上比做多更难**：亏损无上限、需要付资金费率、面对做市商和项目方的逆向操作
- **五层过滤不保证成功**：所有层都通过也可能亏钱——市场有你不知道的信息
- **做市商判断可能错误**：链上数据有延迟，CEX操作不留痕迹，做市商随时可以变向
- **历史案例不等于未来**：Luna的信号在下一次崩溃中可能完全不同
- **这个框架假设你有纪律**：如果你不设止损、不控制仓位，再好的框架也救不了你
- **散户做空的最优策略可能是不做空**：如果你不确定，最好的空头仓位是零
