## 对比总览

| 维度 | 中国象棋 | 国际象棋 | 围棋 | 将棋 | 五子棋 | 军棋 | 跳棋 |
|------|---------|---------|------|------|-------|------|------|

## 中国象棋（Xiangqi）

### 棋盘与布局
- 9列×10行，中间"楚河汉界"
- 九宫格（3×3）限制将/帅和士/仕活动范围

### 棋子与走法
| 棋子 | 红方/黑方 | 走法 |
|------|----------|------|
| 帅/将 | 帅/将 | 九宫内一步直行；不可与对方帅将面对面（"对面笑"） |
| 仕/士 | 仕/士 | 九宫内斜走一步 |
| 相/象 | 相/象 | 斜走两格（"田"字）；不可过河；可被"塞象眼" |
| 車/车 | 車/车 | 直线任意格数（同国际象棋车） |
| 馬/马 | 馬/马 | "日"字走法；可被"蹩马腿" |
| 炮/砲 | 炮/砲 | 移动同車；**吃子必须隔一子跳吃**（独特机制） |
| 兵/卒 | 兵/卒 | 未过河：前进一步；过河后：可前、左、右（不可后退） |

### 核心规则
- 将军(将)：攻击对方将/帅
- 将杀：无法解除将军 → 输
- 困毙：无合法走法 → 输（与国际象棋逼和不同）
- 长将/长捉判负

## 国际象棋（Chess）

### 棋子与走法
| 棋子 | 走法 | 特殊规则 |
|------|------|---------|
| King | 任意方向一格 | 王车易位(Castling) |
| Queen | 任意方向任意格数 | — |
| Rook | 直线任意格数 | 参与王车易位 |
| Bishop | 斜线任意格数 | — |
| Knight | "L"形（2+1）跳跃 | 唯一可跳过其他棋子的棋子 |
| Pawn | 前进一格(首次可两格)；斜前方吃子 | 升变(Promotion)、吃过路兵(En passant) |

### 关键特殊规则
- **王车易位**: 王向車方向走两格，車跳到王的另一侧；条件：王和車都未动过、路径无子、王不经过被攻击的格子
- **吃过路兵**: 对方兵首次走两格经过你的兵可攻击的格子时，可在下一步斜向吃掉
- **升变**: 兵到达底线必须升变为后/車/象/马
- **逼和(Stalemate)**: 无合法走法但未被将军 → 和棋
- **三次重复局面 / 50步规则**: 和棋条件

### 计分系统（棋子价值参考）
| 棋子 | 相对价值 |
|------|---------|
| Queen | 9 |
| Rook | 5 |
| Bishop | 3.25 |
| Knight | 3 |
| Pawn | 1 |

## 围棋（Go / Weiqi / Baduk）

### 基本规则
- 19×19棋盘（361个交叉点），也可用9×9或13×13
- 黑先白后，交替落子
- 规则极简但策略极深

### 核心概念
| 概念 | 说明 |
|------|------|
| 气(Liberty) | 棋子/棋群相邻的空交叉点数 |
| 提子(Capture) | 对方棋子气被完全堵死则提走 |
| 打劫(Ko) | 禁止立即取回形成无限循环 |
| 目(Territory) | 被一方围住的空交叉点 |
| 地(Area) | 活棋+围住的空点 |

### 计分方法
| 规则体系 | 计算方式 | 贴目 |
|---------|---------|------|
| 中国规则(数子法) | 活子+目 | 贴3又3/4子(7.5目) |
| 日本规则(数目法) | 目-提子 | 贴6.5目 |
| 韩国规则 | 同日本 | 贴6.5目 |
| AGA规则 | 两种皆可 | 7.5目 |

### 复杂度
- 合法局面数: ~2.08×10^170（远超国际象棋）
- 平均每步可选约250步（国际象棋约35步）
- 2016年AlphaGo首次击败人类顶尖棋手

## 将棋（Shogi）

### 独特机制——打入(Drop)
- 被吃掉的棋子归对手所有，可在之后的回合作为自己的棋子放回棋盘
- 这意味着棋子永远不会退出游戏，几乎所有对局都以将杀结束

### 棋子与升变
- 9×9棋盘，每方20枚棋子
- 大部分棋子进入对方三行后可升变（翻面显示升变后的走法）
- 玉将（王）不可升变

## 五子棋（Gomoku / 五子棋 / Wuziqi）

### 基本规则
- 15×15棋盘（也可用围棋盘）
- 黑先白后
- 先连成5子（横/竖/斜）者胜

### 竞赛规则（连珠/Renju）
由于黑棋先手优势过大，竞赛中对黑棋有限制：
| 规则 | 说明 |
|------|------|
| 禁手 | 黑棋不可走出三三、四四、长连（6+子） |
| 白棋无禁手 | 白棋长连也算赢 |
| 开局规则 | 指定开局（如Soosyrv-8/Taraguchi-10等） |

## 军棋（Luzhanqi / 陆战棋）

### 棋盘
- 专用地图：道路(细线) + 铁路(粗线) + 山界 + 营地(安全区) + 大本营
- 铁路上可沿直线移动任意格数

### 棋子等级（高到低）
| 等级 | 棋子 | 数量 | 特殊规则 |
|------|------|------|---------|
| 最高 | 司令 | 1 | — |
| 9 | 军长 | 1 | — |
| 8 | 师长 | 2 | — |
| 7 | 旅长 | 2 | — |
| 6 | 团长 | 2 | — |
| 5 | 营长 | 2 | — |
| 4 | 连长 | 3 | — |
| 3 | 排长 | 3 | — |
| 2 | 工兵 | 3 | 可拐弯走铁路；可挖地雷 |
| 特殊 | 炸弹 | 2 | 与任何棋子同归于尽 |
| 特殊 | 地雷 | 3 | 不可移动；除工兵外碰到即同归于尽 |
| — | 军旗 | 1 | 不可移动；被夺则输 |

### 核心机制
- **暗棋**: 棋子竖立朝向自己，对手看不到
- **裁判/第三方**: 两人对战时需裁判判定大小
- **营地安全**: 在营地中的棋子不可被攻击
- **布阵规则**: 炸弹不可放第一行；地雷只能放最后两行；军旗必须在大本营

### 胜负判定
- 夺取对方军旗 → 胜
- 对方无棋可动 → 胜

## 跳棋（Chinese Checkers）

### 规则
- 六角星棋盘，2-6人
- 每人10枚棋子，起始于一个角
- 目标：全部棋子移至对角三角区

### 移动方式
- 单步：移至相邻空位
- 跳跃：跳过相邻棋子到对面空位（可连续跳跃）
- 跳跃不吃子（与西洋跳棋不同）
