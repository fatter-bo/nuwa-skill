## 国标麻将（Chinese Official / MCR）

### 特征
- 1998年国家体委统一制定的竞赛规则
- 144张牌：万/条/筒各36 + 风牌16 + 箭牌12 + 花牌8
- 81种番型，最低8番起胡
- 最大番型：大四喜(88番)、大三元(88番)、九莲宝灯(88番)
- 国际正式比赛采用

### 番型分层
| 番数 | 典型番型 |
|------|---------|
| 88番 | 大四喜、大三元、绿一色、九莲宝灯、四杠 |
| 64番 | 小四喜、小三元、字一色、四暗刻 |
| 48番 | 一色四同顺、一色四节高 |
| 32番 | 一色四步高、三杠、混幺九 |
| 24番 | 七对、清一色、一色三同顺 |
| 16番 | 三风刻、一色三步高、清龙 |
| 8番 | 混一色、推不倒、妙手回春 |
| 6/4/2/1番 | 各种小番型 |

## 广东麻将（Cantonese / Hong Kong Style）

### 核心规则
- 136张牌（无花牌）
- 最流行的"鸡胡"规则几乎无番缚限制
- "推倒胡"：任何合法胡牌皆可
- 自摸通常翻倍计算

### 特色机制
- **抢杠胡**: 对手明杠时可截胡
- **十三幺**: 万筒条各1和9 + 全部字牌 + 任一重复 = 最大牌型
- **杠上花**: 摸杠后补牌即胡
- **海底捞月**: 最后一张牌自摸胡

### 常见番型
| 番型 | 说明 | 番数(参考) |
|------|------|----------|
| 平胡 | 全顺子+一对将 | 1 |
| 对对胡 | 全刻子 | 3 |
| 清一色 | 单一花色 | 7 |
| 混一色 | 单花色+字牌 | 3 |
| 十三幺 | 特殊 | 13 |

## 四川麻将（Sichuan Bloody）

### 核心特征——血战到底
- 仅108张牌：万/条/筒各36，**无字牌、无花牌**
- **不可吃牌**，只能碰和杠
- **缺一门**: 开局必须选择不要的一门花色并全部打出
- **血战到底**: 一人胡牌后游戏继续，直到剩余玩家都胡或流局
- **查叫/查花猪**: 流局时未听牌或未缺一门者罚分

### 计分
- 基础分 × 2^(番数)
- 自摸三家付，点炮放炮者付
- 杠上开花、杠上炮等加番
- 刮风下雨（杠牌即时结算）

### 特殊规则
| 规则 | 说明 |
|------|------|
| 刮风(明杠) | 其余三家各付1倍底 |
| 下雨(暗杠) | 其余三家各付2倍底 |
| 血战到底 | 持续到三人胡或流局 |
| 换三张 | 开局与对家交换3张同花色牌 |

## 武汉麻将（开口翻/红中赖子杠）

### 核心特色
- 红中为万能"赖子"（癞子），可替代任意牌
- 翻一张牌确定赖子（如翻出5万→6万为赖子）
- 可以**开口翻**——翻出的赖子指示牌决定额外玩法
- 杠牌频繁，明杠/暗杠/赖子杠各有不同计分

### 特殊机制
- **赖子杠**: 4张赖子组成的杠，非常稀有
- **抢杠胡**: 对手明杠时可截胡
- **硬胡**: 不使用赖子完成胡牌，分数更高

## 日本麻将（Riichi / 立直麻雀）

### 核心规则
- 136张牌（万/索/筒各36 + 字牌28）
- 严格的"役"（yaku）系统：至少1翻役才能胡牌

### 独有机制
| 机制 | 说明 |
|------|------|
| 立直(Riichi) | 门前听牌时宣言立直，押1000点棒，锁定手牌 |
| 宝牌(Dora) | 翻开指示牌，下一张为宝牌；每含1张+1翻 |
| 里宝牌(Ura-Dora) | 立直胡牌后翻开的额外宝牌 |
| 一发(Ippatsu) | 立直后一巡内胡牌+1翻 |
| 振听(Furiten) | 自己打过的牌组成的待牌不可荣和 |

### 计分体系
- 翻数(Han) × 符数(Fu) 查表确定点数
- 满贯(Mangan, 5翻) = 8000/12000点
- 跳满(Haneman, 6-7翻) = 12000/18000
- 倍满(Baiman, 8-10翻) = 16000/24000
- 三倍满(Sanbaiman, 11-12翻) = 24000/36000
- 役满(Yakuman, 13翻+) = 32000/48000

### 常见役
| 役 | 翻数 | 条件 |
|----|------|------|
| 立直 | 1 | 门前听牌宣言 |
| 门前清自摸 | 1 | 门前清自摸胡 |
| 平和 | 1 | 全顺子+两面听+役牌外雀头 |
| 断幺九 | 1 | 无1、9、字牌 |
| 一盃口 | 1 | 两组相同顺子 |
| 七对子 | 2 | 七个对子 |
| 混一色 | 3(副露2) | 单花色+字牌 |
| 清一色 | 6(副露5) | 单花色 |
| 国士无双 | 役满 | 十三种幺九牌各一+一重复 |

## 台湾麻将（Taiwanese）

### 特征
- 144张牌（含花牌）
- 16张麻将（每人起始16张而非13张）
- 花牌得分独立计算
- 台数制计分
- 可选"连庄"规则：庄家胡牌续庄+加台

### 独特规则
- 花牌自动补牌，积分直接计算
- "抢杠"机制
- 较为灵活的胡牌条件
- 射月（只赢特定牌型时的高额奖励）
