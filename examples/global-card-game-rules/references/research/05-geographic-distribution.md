# 05 地域分布图谱——全球棋牌流行度与蓝海识别

## 全球棋牌地域分布

### 亚洲

| 国家/地区 | 最流行玩法 | 特色变体 | 市场规模特征 |
|----------|----------|---------|------------|
| 中国大陆 | 麻将(各地方言版)、斗地主、掼蛋 | 四川麻将、武汉麻将、广东麻将、跑得快、升级 | 全球最大棋牌市场；移动端主导 |
| 日本 | 麻雀(Riichi)、将棋、花札(Hanafuda) | 立直麻雀独有的Dora/振听系统 | 职业联赛体系完善；Mリーグ等 |
| 韩国 | 围棋(Baduk)、花投(Hwatu)、Go-Stop | 韩式花牌基于日本花札 | 围棋全球最高人均参与率 |
| 印度 | Teen Patti、Rummy(印度拉米) | Teen Patti变体极多(Muflis/AK47等) | 在线Rummy市场>$20亿(2025) |
| 东南亚 | 麻将(新马港式)、Big Two(锄大弟) | 新加坡/马来西亚式麻将 | 社交赌博文化浓厚 |
| 台湾 | 16张台湾麻将 | 含花牌、台数制 | 春节文化标配 |

## 蓝海玩法识别

### 方法论
蓝海 = 有大量本地玩家基数 + 尚未被数字化/全球化的玩法

### 识别出的蓝海机会

| 玩法 | 地区 | 蓝海理由 | 数字化程度 |
|------|------|---------|----------|
| **掼蛋** | 中国(尤其安徽/江苏) | 2023-2025爆发式增长，从地方走向全国，竞赛化程度高 | 中等(腾讯等在做) |
| **Truco** | 巴西/阿根廷 | 拉美人口红利，本地化数字产品少 | 低 |
| **Okey** | 土耳其/中亚 | 土耳其式Rummikub，文化粘性极强 | 中低 |
| **Belote** | 法国/比利时 | 法语区国民游戏，高DAU潜力 | 中等 |
| **Durak** | 俄罗斯/前苏联 | 覆盖2亿+人口，规则简单易上手 | 中等 |
| **Skat** | 德国 | 竞赛体系成熟，玩家忠诚度高 | 中低 |
| **花札/花投** | 日本/韩国 | 文化符号强烈，跨媒体IP潜力(鬼灭之刃) | 低 |
| **Basra** | 埃及/中东 | 中东棋牌市场增长最快 | 极低 |
| **四川麻将出海** | 全球华人社区 | 规则简单、节奏快、适合移动端 | 中等(出海不足) |

### 蓝海评估矩阵

```
          高文化粘性
              │
   Belote ●  │  ● 掼蛋
   Skat ●    │      ● 四川麻将
              │
──────────────┼──────────────
  低数字化    │    高数字化
              │
   Basra ●   │
   Truco ●   │  ● Durak
   花投 ●    │
              │
          低文化粘性
```
