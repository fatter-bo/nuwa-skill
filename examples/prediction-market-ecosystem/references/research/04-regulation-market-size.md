# 04-regulation-market-size: 监管格局与市场规模

## CFTC监管框架

### 分类的历史意义
CFTC将预测市场事件合约归类为**衍生品（swaps）**——这是决定性的法律定位：
- 联邦金融监管，非州赌博法
- 预测市场运营者按DCM（指定合约市场）监管
- 用户保护和市场完整性要求适用

### 2025-2026关键监管动作
| 时间 | 事件 |
|------|------|
| 2025 | DCM认证~1,600个事件合约 |
| 2025.07 | Polymarket注册CFTC |
| 2025.11 | Polymarket获DCM修正批准 |
| 2026.02 | CFTC执法部门发布预测市场非公开信息滥用咨询 |
| 2026.03 | CFTC发布ANPRM + DCM工作人员咨询函（评论期至2026.04.30） |
| 2026.03 | CFTC在联邦法院确认对预测市场的独占管辖权 |

### 双重监管现实
- 联邦层面：CFTC合法化
- 州层面：多州博彩委员会反对（认为侵犯其管辖权）
- 赛马行业：根据Interstate Horseracing Act反对赛马预测合约
- 张力未完全解决

### 全球监管
- 英国：Betfair作为持牌博彩交易所自由运营
- 欧盟：各国不同——有的分类为赌博，有的分类为金融工具
- 离岸DeFi：Polymarket非美国版无地域监管锚点
- VPN绕过现实：IP筛查+设备指纹+KYC，但假名钱包使执法困难

### FastEvent的监管定位
**数据层不直接受赌博监管**：
- 提供数据≠运营市场
- 类比Sportradar——数据供应商不是博彩运营商
- 合规预测市场反而更需要可审计数据源
- 链上记录满足审计要求

## 市场规模

### 全球预测市场
| 年份 | 交易量 | 增长 |
|------|--------|------|
| 2024 | ~$90亿 | — |
| 2025 | $440亿+ | ~5x |
| 2026.01月 | $200亿+ | — |
| 2026.02月 | $168亿（Poly $70亿 + Kalshi $98亿） | — |
| 2026年化保守 | $2000亿+ | ~5x |
| 2030预测 | $1.1万亿 | — |

### 月活数据
- 2026.02月活钱包：840,000
- 行业周交易量预测（Keyrock/Dune）：$250亿/周 = $1.3万亿/年

### 体育品类占比
- Kalshi：~90%交易量来自体育
- Polymarket：体育为最大品类（2025.02起）
- 全球体育博彩TAM：$1190亿（2025），$1250亿（2026）

### FastEvent可触达市场推导
```
预测市场2026年化: ~$2000亿+
体育占比: ~60-90%
= 体育预测市场: ~$1200-1800亿
FastEvent作为数据层take rate: 假设0.01-0.05%
= FastEvent潜在收入: $1200万-$9000万/年
```
