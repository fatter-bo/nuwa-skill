# 研究维度六：多市场对比分析

## 1. 多市场数据对比方法论

### 对比框架

同一游戏在不同国家/地区上线后，应建立标准化的多市场对比分析框架。

**Step 1：建立对比基线**
- 选定"基准市场"（通常是最早上线或数据最充分的市场）
- 所有市场指标对齐到相同的安装后天数（D1/D7/D30等）
- 确保Cohort口径一致（同一时间窗口、同一版本）

**Step 2：标准化指标对比**

| 对比维度 | 指标 | 为什么重要 |
|---------|------|-----------|
| 留存质量 | D1/D7/D30留存 | 产品是否适合该市场 |
| 变现效率 | ARPDAU/ARPU/eCPM | 该市场的付费能力 |
| 获客效率 | CPI/CAC/ROAS | 获取用户的成本效率 |
| 参与深度 | Session时长/次数/日均局数 | 用户的投入程度 |
| 病毒性 | K-Factor/自然量占比 | 口碑传播效果 |
| 技术适配 | 崩溃率/加载时间 | 技术基础设施匹配 |

**Step 3：归因差异来源**
差异来源通常为三类：
1. **市场特性**：付费能力、文化偏好、竞争环境（不可改变，只能选择）
2. **产品适配**：本地化质量、内容偏好匹配度（可优化）
3. **运营执行**：UA策略、LiveOps适配、渠道选择（可优化）

**来源**：Sensor Tower "State of Mobile Gaming 2025"; Udonis "200+ Mobile Gaming Market Statistics"; Airbridge "Global Game Growth 2025"
**可信度**：高 — 行业顶级数据分析平台方法论

## 2. 继续投入 vs 放弃的决策框架

### Go/Scale/Optimize/Kill 四级决策

```
市场评估矩阵：

         留存达标（D1>30%且D7>10%）     留存不达标
        ┌─────────────────────────┬──────────────────────────┐
ROAS    │ ★ SCALE                 │ OPTIMIZE（产品优先）       │
达标    │ 加大投放预算              │ 留存问题 → 先优化产品     │
        │ 扩展渠道覆盖             │ 再重新测试ROAS            │
        ├─────────────────────────┼──────────────────────────┤
ROAS    │ OPTIMIZE（变现优先）      │ ✕ KILL / HOLD            │
不达标  │ 留存OK → 优化变现/CPI    │ 留存+变现双差             │
        │ 降低CPI或提升ARPDAU     │ 放弃或大幅改版后重测       │
        └─────────────────────────┴──────────────────────────┘
```

### 量化决策标准

| 信号 | 判定 | 行动 |
|------|------|------|
| D1留存 < 25% | RED | 暂停投放，优化FTUE |
| D1留存 25-35%且D7 < 8% | YELLOW | 小规模测试，产品优化中 |
| D1 > 35%且D7 > 10%且D7 ROAS > 目标 | GREEN | 放量 |
| LTV/CPI < 0.8（且无优化空间） | RED | 放弃该市场 |
| LTV/CPI 0.8-1.2 | YELLOW | 优化CPI或变现 |
| LTV/CPI > 1.5 | GREEN | 积极放量 |

### 市场退出的定量信号

当以下条件**同时满足**时，考虑退出某一市场：
1. D7 ROAS持续低于目标50%以上
2. 经过2轮以上A/B测试优化后无显著改善
3. CPI持续上升且无创意突破
4. 该市场LTV/CPI < 0.7
5. 资源投入该市场的机会成本（对比其他市场）过高

## 3. 各国行为特征差异的数据表现

### 收入与变现差异

**收入Top市场**（2025）：
- 中国：$36,890M（最大市场）
- 美国：紧随其后
- 日本、韩国：Top 3-4

**IAP增长趋势**：
- 北美：+9% YoY
- 欧洲：+14% YoY
- 中东：+18% YoY（增速最快）
- 拉美：+13% YoY

**eCPM差异**：
- 美国：Rewarded Video $12.91（Android）/$13.18（iOS）
- Top 20国家Android：$3.31-$12.91
- Top 20国家iOS：$4.73-$13.81
- T3国家：通常低于$3

### 各区域用户行为特征

**北美市场**：
- 数据表现：ARPU高（~$60/年），eCPM全球最高，CPI也最高
- 用户特征：IAP和广告接受度均高，对Rewarded Video尤其友好
- 棋牌偏好：Texas Hold'em、Gin Rummy为主
- 平台分布：iOS/Android各50%

**日本/韩国市场**：
- 数据表现：ARPU极高，但CPI也极高，eCPM仅次于欧美
- 用户特征：付费意愿强但挑剔，品质要求高，留存通常高于欧美
- 棋牌偏好：日本麻将、将棋、花札；韩国Go-Stop、麻将
- 平台分布：日本iOS主导65%+；韩国Android主导

**东南亚市场**：
- 数据表现：CPI极低（$0.1-0.5），但ARPU也低，eCPM处于T3水平
- 用户特征：Session长但付费转化低，广告容忍度高
- 棋牌偏好：因国家差异极大（印尼棋牌、越南Tien Len、泰国Dummy）
- 平台分布：Android主导80%+
- 注意：需适配低端设备和不稳定网络

**中东/北非市场**：
- 数据表现：增速最快（+18% YoY），海湾国家ARPU接近欧美
- 用户特征：斋月期间游戏时长大幅增加，是全年最重要营销窗口
- 棋牌偏好：Tarneeb、Baloot为主流
- 注意：RTL适配、阿拉伯语本地化质量

**欧洲市场**：
- 数据表现：ARPU中-高，eCPM增长强劲（德国表现最佳，广告格式全面增长）
- 用户特征：多语言挑战（至少英/法/德/西/意/葡6种），GDPR合规要求
- 棋牌偏好：因国别差异大（德国Skat、法国Belote、西班牙Mus）
- 平台分布：iOS/Android接近均衡

**拉美市场**：
- 数据表现：CPI低，ARPU低-中，增速中等（+13% YoY）
- 用户特征：社交分享意愿高，K-Factor潜力大
- 棋牌偏好：Truco（巴西）为代表

**可信度**：高 — 多个权威数据平台交叉验证

## 4. 多市场对比分析的实操流程

### Step-by-Step执行清单

```
□ 1. 数据准备
   - 按国家拉取同期Cohort数据
   - 确保版本一致（或标注版本差异）
   - 统一货币单位（建议USD）

□ 2. 基础指标对比仪表盘
   - 留存矩阵：各国 × D1/D7/D30
   - 变现矩阵：各国 × ARPDAU/eCPM/付费率
   - 获客矩阵：各国 × CPI/ROAS/自然量占比

□ 3. 差异诊断
   - 标注显著偏离基准市场的指标（>20%差异）
   - 按差异来源分类（市场特性/产品适配/运营执行）
   - 对可优化项制定行动计划

□ 4. 优先级排序
   - 用ROI潜力排序：预期收益 × 成功概率 / 投入资源
   - 优先优化"留存好但变现低"的市场（提升变现更容易）
   - 其次优化"变现好但留存低"的市场（需要产品调整）

□ 5. 决策输出
   - SCALE：加大投放的市场列表
   - OPTIMIZE：需要优化的市场+具体方向
   - HOLD：观望的市场+重测条件
   - KILL：放弃的市场+原因记录
```

## 5. 跨平台对比注意事项

同一国家内iOS和Android的数据可能差异巨大：
- 日本Top 5游戏在iOS和Android的排行可能完全不同
- iOS用户通常ARPU更高但CPI也更高
- Android用户量更大但付费转化更低
- 建议按平台分别分析，不要简单合并

**可信度**：高 — 行业权威报告
