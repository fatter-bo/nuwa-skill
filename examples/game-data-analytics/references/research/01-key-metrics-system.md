# 研究维度一：关键指标体系

## 1. 留存指标体系

### D1/D3/D7/D14/D30留存基准

留存率是衡量游戏产品健康度的核心温度计。不同品类的留存基准差异显著。

**休闲游戏（Puzzle/Merge/合成类）留存基准**：
- D1：35-45%（Top25%达45%+，Top1%可达64-68%）
- D7：12-18%（中位数约3.4-3.9%，Top25%达7-8%）
- D14：8-12%
- D30：6-10%（中位数约5.35%，Top tier达9%+）

**棋牌游戏（Casino/Tabletop/Card）留存基准**：
- D1：35-40%（棋牌用户粘性高，但首次体验要求也高）
- D7：15-20%
- D30：4.10-5.51%中位数，Top tier达8%+
- 棋牌品类特点：session length几乎是其他品类的2倍，长期留存受社交绑定影响大

**超休闲留存基准**：
- D1：20-30%
- D7：3-5%
- D30：<2%

**行业总体中位数**（GameAnalytics 2025报告）：
- D1约22%（Top10%达40%）
- D7约3.4-3.9%（Top25%达7-8%）
- D28：75%的项目低于3%

**来源**：GameAnalytics 2025 Mobile Gaming Benchmarks Report; Game Growth Advisor "The 20 Mobile Game KPIs That Actually Matter in 2026"
**可信度**：高 — 基于16K+活跃移动游戏的统计数据

### 留存率计算方式
- **经典留存（Classic Retention）**：Day N当天回来的用户 / 当天安装用户
- **滚动留存（Rolling Retention）**：Day N及之后任意一天回来的用户 / 安装用户
- **区间留存（Range/Bracket Retention）**：Day N到Day M之间回来的用户 / 安装用户

## 2. 活跃与粘性指标

### DAU/MAU与粘性比

- **DAU/MAU Stickiness Ratio**：衡量用户每月有多少天回来
- 休闲游戏目标：20%+为Good，30%+为World-class
- 低于7%表示产品存在严重问题
- 棋牌游戏通常粘性更高，因为社交对战和排名系统驱动日常回访

### Session指标

- **Session长度**：中位数5-6分钟，Top25%达8-9分钟
- **棋牌特殊性**：Session长度几乎是其他品类的2倍（因为单局时间长）
- **日均Session次数**：平均4次/天，Top1%达12-14次
- **日均游戏时长**：中位数约22分钟，Top2%可达4小时

**来源**：GameAnalytics 2025 Mobile Gaming Benchmarks
**可信度**：高 — 行业权威数据源

### 棋牌专属活跃指标

- **场均局数**：单次Session内的对局次数，反映单次留存深度
- **日均局数**：每日完成的对局数，棋牌核心参与指标
- **等级场迁移率**：从低等级场向高等级场迁移的用户比例
- **Bot占比**：AI机器人在对局中的占比，影响匹配体验和真实数据

## 3. 变现指标

### ARPU/ARPPU/ARPDAU

- **ARPDAU**（Average Revenue Per Daily Active User）：
  - 广告变现型：$0.05-$0.10典型，$0.15-$0.20+为强
  - IAP驱动型：$0.15-$0.30典型，$0.50-$1.00+为强
  - 混合变现型：$0.10-$0.20典型，$0.25-$0.50+为强
- **月ARPU**：休闲游戏$1-$5，中核$5-$15
- **付费转化率**：2%以上为强，Top游戏3-5%，社交棋牌可超8%
- **ARPPU**：付费用户平均收入，反映付费深度

### eCPM

- eCPM范围：$2-$50+/千次展示，取决于广告格式和地区
- 美国Rewarded Video：iOS $13.18，Android $12.91
- 美国Interstitial：iOS $14.32，Android $14.08
- Banner：低于$1/千次展示
- 区域排序：欧美 > 港澳台 > 日韩 > 俄罗斯 > T3国家 > 东南亚/南亚/拉美

**来源**：Tenjin Ad Monetization Benchmark Report 2025; TopOn H1 2025 Global Mobile Games Monetization Report
**可信度**：高 — MMP和广告聚合平台一手数据

### LTV

- **超休闲D180 LTV**：$0.50-2.00
- **休闲D180 LTV**：$3.00-15.00
- **中核休闲D180 LTV**：$8.00-25.00
- **合成游戏头部D180 LTV**：$15.00-40.00
- **美国市场ARPU**（2026预估）：约$60.58

### 棋牌专属变现指标

- **欢乐豆/筹码流通量**：虚拟货币日产出与消耗的平衡
- **胜率分布**：理想状态下接近正态分布，过偏表示匹配或Bot问题
- **付费场渗透率**：从免费场到付费场的转化比例

## 4. 获客指标

- **CPI**（2026基准）：
  - 休闲iOS $2.50 / Android $1.50
  - 中核iOS $4.50 / Android $3.25
- **ROAS目标**：D7 15-25%，D30 40-60%
- **自然量倍数**：健康范围1.5-2.0x
- **K-Factor**：>1.0表示病毒式增长

## 5. 技术指标

- **崩溃率**：目标<1%，Top工作室达0.5%以下
- **加载时间**：首次<5秒，后续<2秒
