# 04-payment-model: 付费模型本地化

## 各国主流付费价格点

### 日本
- **主流价格点**：120円($0.99)、610円($4.99)、3,060円($24.99)、9,800円($79.99)
- **特征**：IAP极高，人均IAP消费$94.3（全球最高之一）
- **热门付费模式**：gacha抽卡、月卡/通行证、限定角色直购
- **福袋文化**：新年"お年玉パック"（福袋礼包）是传统高峰

### 韩国
- **主流价格点**：与日本类似的阶梯结构
- **特征**：RPG品类IAP占80-90%收入
- **热门付费模式**：装备强化石、VIP特权、赛季通行证
- **支付**：信用卡、KakaoPay、Naver Pay

### 中国
- **主流价格点**：6元($0.99)、30元($4.49)、68元($9.99)、198元($28.99)、648元($99.99)
- **特征**：月卡($4.99级别)渗透率极高
- **热门付费模式**：首充双倍、月卡、通行证、限时折扣
- **关键**：648元是iOS单笔上限，也是"大R"消费的基础单位

### 东南亚
- **特征**：低ARPU但高增长（泰国+16%、印尼增长强劲）
- **定价策略**：需大幅下调，$0.99起步太贵
- **热门付费模式**：小额高频IAP ($0.10-$1.00)、rewarded ads
- **关键**：信用卡渗透率低，运营商代扣(carrier billing)是关键支付方式

### 印度
- **特征**：全球下载量TOP3但ARPU极低
- **定价策略**：$0.10-$0.50的微交易
- **支付**：UPI (Unified Payments Interface)是主导支付
- **广告为王**：IAA收入 >> IAP收入

### 美国/欧洲
- **主流价格点**：$0.99、$4.99、$9.99、$19.99、$49.99、$99.99
- **特征**：Battle Pass / Subscription模式流行
- **热门付费模式**：cosmetic直购、Battle Pass、订阅
- **反P2W**：纯装饰性付费接受度最高

### 中东/MENA
- **特征**：人均消费增长18%（2024 YoY）
- **高端玩家突出**："鲸鱼"群体消费力强
- **支付**：预付卡、Apple Pay、本地电子钱包

## IAA vs IAP 变现矩阵

| 品类 | IAA占比 | IAP占比 | 最佳策略 |
|------|--------|--------|---------|
| 超休闲(Hyper-casual) | 90%+ | <10% | 纯广告驱动 |
| 休闲(Casual) | 40-60% | 40-60% | 混合变现 |
| 中核(Mid-core) | 15-25% | 75-85% | IAP主导+rewarded ads |
| 硬核(Hardcore) | <10% | 90%+ | IAP/订阅为主 |
| 文字/益智 | 50-70% | 30-50% | 混合偏广告 |

**2025趋势**：72%的开发者采用混合变现模式。纯单一变现方式正在减少。

## 当地支付方式

| 地区 | 主要支付方式 | 必须集成 |
|------|-----------|---------|
| 中国 | 支付宝、微信支付 | 必须。Apple Pay/Google Pay在iOS/Android内已覆盖 |
| 日本 | 信用卡、便利店支付、运营商代扣 | 便利店支付(Konbini)覆盖无信用卡用户 |
| 韩国 | 信用卡、KakaoPay、Naver Pay、T-money | KakaoPay生态覆盖广 |
| 东南亚 | GrabPay、GCash、OVO、Dana、运营商代扣 | 运营商代扣(DCB)最关键 |
| 印度 | UPI、Paytm、PhonePe | UPI是绝对主流 |
| 巴西 | PIX、Boleto、信用卡 | PIX即时支付已成标准 |
| 中东 | Apple Pay、预付卡、STC Pay | 预付卡覆盖无银行账户人群 |
| 俄罗斯/独联体 | QIWI、YooMoney | 因制裁需特殊处理 |

## 本地化定价策略

### 购买力平价(PPP)调整
本地化IAP定价后，收入可提升20%-40%。

**定价公式参考**：
```
本地价格 = 基准价格(USD) × 汇率 × PPP调整系数 × 心理价格圆整
```

### 心理价格策略
- **美国**：$X.99结尾（$4.99、$9.99）
- **日本**：整数居多（610円、3,060円）
- **中国**：吉利数字结尾（68元、88元、168元、648元）
- **东南亚**：超低起步价，高频小额

### 首充/新人礼包
- **全球通用**：首次付费给予2-3倍价值
- **中国特色**：首充双倍是标配，不做会被认为不大方
- **日本特色**：新人期间限时"初心者パック"
- **欧美**：Starter Pack + Battle Pass组合
