# 05-compliance-regulation: 法规与合规

## 隐私法规全景

### GDPR（欧盟/欧洲经济区）
- **适用范围**：处理EU/EEA居民数据的所有公司，无论公司所在地
- **未成年保护**：16岁以下需父母同意（成员国可降至13岁）
- **关键要求**：
  - 明确的数据收集同意（opt-in）
  - 数据最小化原则
  - 被遗忘权（用户可要求删除数据）
  - 数据可携带权
  - 72小时内数据泄露通知
  - 数据保护影响评估(DPIA)
- **罚款**：最高2000万欧元或全球营收4%
- **游戏影响**：玩家行为数据收集、广告追踪、分析SDK都需要明确同意

### COPPA（美国）
- **适用范围**：面向13岁以下儿童的在线服务
- **2025年修订（2026.04生效）**：
  - 扩大数据收集披露要求
  - 更严格的可验证父母同意(VPC)标准
  - 加强儿童信息安全义务
  - 限制针对未成年人的定向广告
  - 强化父母访问和删除子女数据的权利
- **罚款**：FTC执法，单次违规可达数百万美元
- **游戏影响**：如果游戏有可能吸引儿童玩家，必须合规

### PIPA（韩国）
- **适用范围**：在韩国收集或处理个人信息的所有实体
- **关键要求**：
  - 实名认证（游戏中极其重要）
  - 严格的数据跨境传输限制
  - 数据泄露通知义务
- **游戏特殊规定**：
  - 宵禁制度（虽已放宽，但社会关注度仍高）
  - 概率型道具必须公示概率

### PIPL（中国）
- **适用范围**：处理中国境内个人信息的所有实体
- **关键要求**：
  - 数据本地化（敏感数据必须存储在中国境内）
  - 跨境数据传输需安全评估
  - 未成年实名认证+防沉迷系统
- **游戏影响**：版号制度、未成年人游戏时间限制

### APPI（日本）
- **适用范围**：处理日本居民个人信息的企业
- **特点**：比GDPR宽松，但2022年修订后加强了跨境数据传输限制
- **游戏影响**：特定电信业务法(JACT)对虚拟货币有额外规定

## 虚拟货币与概率公示

### 中国
- **概率公示**：2017年起强制要求公示所有随机道具的掉落概率
- **每日限购**：部分品类有每日购买上限
- **渐进概率**：要求概率随购买次数逐渐提高（对玩家有利）
- **虚拟货币**：不得与人民币直接挂钩，需明确虚拟货币与现实货币的转换关系

### 日本
- **Kompu Gacha（コンプガチャ）**：2012年起违法——禁止"收集特定组合才能获得奖励"的机制
- **概率公示**：行业自律要求（JOGA/CESA指南），非法律强制
- **特定商取引法**：虚拟货币交易需遵守消费者保护法
- **资金决済法**：预付式支付手段的发行需要备案

### 韩国（GRAC）
- **概率公示**：法律强制要求——《游戏产业振兴法》施行令
- **指导方针**：GRAC发布"概率信息公示指南"
- **合规率**：2025年研究显示合规率显著提高
- **处罚**：不合规可导致评级撤销

### 比利时
- **最严格**：任何付费loot box被认定为非法赌博
- **结果**：EA被罚款；多款游戏在比利时下架loot box功能
- **当前状态**：完全禁止含真实货币购买的随机奖励机制

### 荷兰
- **标准**：loot box中的物品如果有市场价值且可在游戏外交易，则违法
- **EA罚款**：FIFA Ultimate Team被罚500万欧元
- **当前状态**：可交易loot box被禁止

### 澳大利亚
- **2024年新规**：含赌博类机制的游戏强制获得至少M分级（不推荐15岁以下）
- **趋势**：可能进一步立法

### FTC执法案例（美国）
- **2025年**：与Cognosphere（原神开发商）达成$2000万和解
  - 未披露稀有道具的实际掉落概率
  - 要求明确公示道具概率
  - 透明化虚拟货币兑换率
  - 简化购买流程

## 年龄分级系统

| 系统 | 覆盖地区 | 等级 | 是否强制 |
|------|---------|------|---------|
| ESRB | 北美 | E, E10+, T, M17+, AO18+ | 行业自律（console强制） |
| PEGI | 欧洲40+国 | 3, 7, 12, 16, 18 | 多数国家法律强制 |
| CERO | 日本 | A(全年龄), B(12+), C(15+), D(17+), Z(18+) | 行业自律（console强制） |
| GRAC | 韩国 | 全年龄, 12+, 15+, 18+ | 法律强制 |
| GSRR | 台湾地区 | 普遍级, 保护级(6+), 辅12级, 辅15级, 限制级(18+) | 法律强制 |
| USK | 德国 | 0, 6, 12, 16, 18 | 法律强制 |
| ACB | 澳大利亚 | G, PG, M, MA15+, R18+, RC | 法律强制 |
| IARC | 全球数字平台 | 基于问卷自动映射到各地区评级 | App Store/Google Play使用 |

### IARC（国际年龄分级联盟）
- 2013年成立，覆盖Google Play、Microsoft Store、Nintendo eShop等
- 开发者填写内容问卷 → 自动生成各地区对应评级
- **极大简化了多地区分级流程**
- 注意：Sony PlayStation和部分地区仍需单独申请

## 各地区特殊合规要求

| 地区 | 特殊要求 |
|------|---------|
| 中国 | 版号制度（需国家新闻出版署审批）、防沉迷系统、未成年人周五六日及节假日每天限玩1小时 |
| 韩国 | 实名认证、概率公示、防沉迷（已放宽但仍有社会监督） |
| 日本 | Kompu Gacha禁止、JOGA自律规范 |
| 德国 | USK强制分级、暴力内容严格审查（已较以前放宽） |
| 澳大利亚 | RC(拒绝分类)=全面禁止销售、性暴力零容忍 |
| 沙特阿拉伯 | GCAM评级、伊斯兰内容审查 |
| 越南 | 需要版号、外资运营需本地合作伙伴 |
