- **总部**：澳大利亚，Curacao牌照
- **估值**：$14B-$23.5B（2025）

### 核心数据
| 指标 | 数据 |
|------|------|
| 2024 GGR | $4.7B（较2022增长80%） |
| 月入金 | $1.1B |
| 月访问量 | 127M+ |
| 月预估投注额 | $10B |
| 注册用户 | 数百万 |

### 运营特点
- **无代币**：纯中心化运营，不发行原生代币
- **名人营销**：Drake（$100M/年）、UFC赞助
- **区块链集成**：仅限加密支付入出金，游戏逻辑中心化
- **争议**：2025年遭RICO诉讼，Drake/Adin Ross异常胜率被Bloomberg调查
- **CFT合规压力**：面临反恐融资合规审查

- 合同价值：$100M/年（Bloomberg估$180M含现金+股权）

- **2025年8月破裂**：Drake公开称Craven为"snake"，声称被阻止提款4次

## Rollbit — GambleFi代币先驱

### 基本信息
- 产品线：赌场 + 体育博彩 + 1000x杠杆加密期货
- 代币：RLB（ERC-20）

### 收入数据
| 指标 | 数据 |
|------|------|
| 月收入 | $18-30M |
| 24小时收入峰值 | $3M+ |
| 日均RLB烧毁 | ~370万枚（~$310K） |

### RLB Buy & Burn机制
- **收入分配比例**：
  - 1000x加密期货：30%收入→回购RLB
  - 体育博彩：20%收入→回购RLB
  - 赌场：10%收入→回购RLB
- **执行方式**：通过Uniswap自动每小时回购（Rollbot.eth）
- **烧毁比例**：回购的RLB中90%永久销毁，10%分配给Rollbot NFT质押者
- **累计烧毁**：3B+ RLB，占总供应量58.67%（截至2025年6月）
- **关键特性**：即使平台当日亏损也执行回购烧毁
