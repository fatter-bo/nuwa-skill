> 科特迪瓦是UEMOA区域最大经济体（GDP约占40%），人口2800万+，移动支付渗透率高。

## 一、LONACI机构画像

### 基本信息

| 项目 | 内容 |
|------|------|
| 全称 | Loterie Nationale de Cote d'Ivoire（LONACI） |
| 成立 | 1970年（法律No. 70-208，1970年3月20日） |
| 法律地位 | 国有控股的股份有限公司（SA a participation financiere publique majoritaire） |
| 股权结构 | 国家80% + CNPS（国家社保基金）15% + 员工持股5% |
| 总部 | 阿比让（Abidjan） |
| 现任DG | Dramane Coulibaly（同时担任非洲彩票协会ALA主席，2024年3月连任） |
| 副DG | Abdoulkarim Ouattara |
| 董事会主席（PCA） | Youssouf Fofana（2024年11月接任，前任Dr. Lenissongui-Coulibaly Wenceslas于2024年5月去世） |
| 网点规模 | 11,000+销售点，带动约25,000间接就业 |

### 财务表现

| 年份 | 营收（FCFA） | 营收（欧元约） | 净利润（FCFA） | 增长率 |
|------|------------|-------------|-------------|--------|
| 2019 | 2000亿+ | ~3.05亿 | - | - |
| 2022 | - | - | - | - |
| 2024 | 5612.42亿 | ~8.55亿 | - | +8.29%（超目标5450亿） |
| 2025 | 6576亿 | ~10亿 | 126亿 | +17.2% |
| 2026目标 | 7443亿 | ~11.4亿 | - | +13.2% |

**关键洞察**：
- 2024年向玩家返还2880亿FCFA，2025年返还约4140亿FCFA
- 资本金将从4亿FCFA增资至50亿FCFA（12.5倍），表明正在为重大投资做准备
- 在科特迪瓦企业中排名第7位
- 2026年定位"增长负责任化"（croissance responsable）

### 与政府关系

- **深度绑定**：国家持股80%，LONACI是政府财政收入重要来源
- **政策工具**：每年通过税收和分红为国家贡献数百亿FCFA
- **社会角色**：通过Masse Commune等项目建设学校等基础设施
- **监管配合**：与ARJH（博彩监管局）紧密合作打击非法博彩

### 国际合作伙伴

| 合作伙伴 | 领域 | 关系状态 |
|----------|------|---------|
| FDJ/LVS | 体育博彩平台（ABP） | 2012年上线，**已被ADMAIORA替代**（2025年1月） |
| ADMAIORA（Flutter子公司） | 新SportCash平台 | 2025年1月启动，现任技术合作伙伴 |
| Afitech CI | 国家博彩集中平台（PPP模式） | 2025年10月启动实时交易监控 |
| PMU France | 赛马博彩内容 | 长期合作，使用法国赛马赛事（如Prix d'Amerique的Quinte+） |

**要点**：LONACI在2025年初完成了体育博彩技术栈的重大切换——从FDJ的LVS/ABP转向Flutter旗下ADMAIORA。这意味着LONACI正在寻求更强的技术合作伙伴，可能对新技术方案持开放态度。

## 四、科特迪瓦博彩监管框架

### 4.1 法律基础

| 法规 | 内容 |
|------|------|
| Law No. 2020-480（2020年5月27日） | 博彩业基本法，确立分类监管体系 |
| LONACI特许经营令 | 授予LONACI彩票、体育博彩、在线游戏的独家特许权 |
| 2023年12月新令 | 加强非法博彩打击力度的新法令 |

### 4.2 监管机构

| 机构 | 角色 |
|------|------|
| ARJH（Autorite de Regulation des Jeux de Hasard） | 博彩监管局，负责发放许可、市场监督 |
| LONACI | 国家彩票运营商，持有排他性特许经营权 |
| Afitech CI | PPP合作方，运营国家级交易监控平台 |

### 4.3 牌照制度

| 制度 | 适用范围 | 进入路径 |
|------|---------|---------|
| 特许经营制（Concession） | 彩票、体育博彩、在线游戏 | **仅LONACI持有**，外部进入须通过LONACI合作 |
| 授权制（Autorisation） | 赌场、角子机 | 可向ARJH申请，但与彩票/博彩分离 |

### 4.4 垄断可否被挑战？

- **当前状态**：LONACI的特许经营权由法律保障，2020年法律进一步巩固
- **第二张牌照可能性**：极低。与塞内加尔（LONASE获准第二平台）不同，科特迪瓦监管逻辑是"巩固LONACI垄断+打击非法"
- **ARJH-LONACI-Afitech三方平台**的建立进一步表明政府选择强化而非开放市场
- **唯一可能的窗口**：以技术供应商/B2B合作伙伴身份进入，而非作为竞争运营商

> **需法律顾问确认**：特许经营权的具体到期时间和续期条件。

### 4.5 税务环境（2025年起）

| 税种 | 税率 |
|------|------|
| 企业所得税（博彩企业） | 30% |
| 奖金预扣税 | 15% |

## 五、非法博彩竞争

### 5.1 现状

- **DG Dramane Coulibaly（2025年2月）**：公开痛斥"reseaux paralleles frauduleux"（欺诈性平行网络），称其为LONACI转型和增长的"重大障碍"
- **年损失估计**：约60亿FCFA（约900万欧元）/年
- **来源**：既有外部非法运营商，也有来自LONACI内部的平行网络
- **ARJH执法**：2024-2025年最大规模打击行动，销毁110+台非法角子机

### 5.2 非法竞争形态

| 类型 | 描述 | 威胁程度 |
|------|------|---------|
| 内部平行网络 | LONACI经销商自行建立的绕过官方系统的博彩网络 | 高（DG专门提及） |
| 非法角子机厅 | 无牌经营的电子游戏/赌博场所 | 中（正在被打击） |
| 离岸在线平台 | 未授权的国际博彩网站面向科特迪瓦用户 | 中（无本地法律保护） |
| 地下彩票 | 传统非正规彩票活动 | 低-中 |

### 5.3 应对措施

- **Afitech实时监控平台**（2025年10月）：国家级交易监控，理论上可终结非法运营
- **联合执法**：LONACI与宪兵、警察、海关联合行动
- **2023年新法令**：强化法律工具

## 六、市场进入分析：三条路径

### 路径对比矩阵

| 维度 | 路径A：与LONACI合作 | 路径B：竞争/独立牌照 | 路径C：并行（技术供应商） |
|------|-------------------|-------------------|----------------------|
| **可行性** | 高 | 极低 | 高 |
| **进入速度** | 6-12个月 | 3-5年（若可能） | 3-6个月 |
| **收入模式** | 技术许可费/收入分成 | 直接运营收入 | B2B合同收入 |
| **法律风险** | 低（在LONACI框架内） | 极高（挑战国家垄断） | 低（不触及博彩运营） |
| **前提条件** | LONACI有技术缺口且愿合作 | 法律改革/第二牌照开放 | 产品有不可替代性 |
| **收入上限** | 中 | 高（理论上） | 中-低 |
| **关系管理** | 需维护长期政府关系 | 需极强政治资源 | 商业关系为主 |
| **推荐度** | 首选 | 不推荐 | 次选（可与A并行） |

### 路径A详解：与LONACI合作

**LONACI的利益诉求**：
1. 数字化转型加速（目标25%数字营收）
2. 打击平行网络的技术手段
3. 新产品品类（缺乏真正快开产品）
4. 降低对单一技术伙伴的依赖（刚从FDJ换到ADMAIORA）

**我方筹码**：
1. 快速事件数据能力（若为FastEvent）
2. 快开产品设计经验
3. 数据分析/风控能力
4. 不威胁LONACI的运营主权

**谈判要点**：
- 以"技术赋能"而非"市场分割"定位
- 入口：填补产品缺口（快开游戏、实时数据）
- 首期项目建议小而快，证明价值后扩大
- 关注DG Dramane Coulibaly的战略重点：数字化+反欺诈

### 路径C详解：技术供应商

**适合场景**：
- 提供数据feed给SportCash（赛事数据、赔率引擎）
- 为Afitech平台提供反欺诈数据分析
- 为Loto Bonheur提供快开产品引擎
