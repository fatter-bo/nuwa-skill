# 03 棋牌游戏特殊合规调研

调研时间：2026-04-12
调研范围：各国"模拟赌博"定义差异、虚拟货币棋牌法律地位、牌照要求、概率公示、loot box监管

---

## 1. "模拟赌博"定义的全球差异

### 核心争议

棋牌游戏是否构成"赌博"，各国判定标准差异极大。关键变量：

| 判定要素 | 说明 |
|----------|------|
| 对价（Consideration） | 用户是否支付了有价值的东西（金钱/虚拟货币） |
| 机会（Chance） | 结果是否主要由运气决定（vs 技巧） |
| 奖品（Prize） | 是否可获得有现实价值的回报 |

三要素同时满足 = 赌博。缺少任一要素，多数司法管辖区不认定为赌博。

### 各国立场对照

| 国家/地区 | 对虚拟货币棋牌的态度 | 关键法规 |
|-----------|---------------------|----------|
| 美国 | 各州差异极大；多数州认为纯虚拟货币不构成"prize" | 各州赌博法 |
| 英国 | UKGC严格监管；虚拟货币若可回流为真金则视为赌博 | Gambling Act 2005 |
| 中国大陆 | 所有涉赌App禁止；虚拟货币棋牌如有"房卡模式"可能被认定为赌博 | 刑法第303条 |
| 日本 | 区分"赌博"和"游技"（如弹珠机）；gacha受景品表示法约束 | 刑法第185条 |
| 韩国 | 网游内赌博活动受限；概率公示为法律强制 | 游戏产业促进法 |
| 德国 | 2021年州际赌博条约后，对模拟赌博分类趋严 | GlüStV 2021 |
| 澳大利亚 | 联邦层面禁止在线赌博提供；含loot box游戏分级提高 | Interactive Gambling Act 2001 |
| 印度 | 各邦差异巨大；技巧型游戏（如Rummy）多数邦合法 | 各邦赌博法 |
| 巴西 | 2024年底通过新框架法，逐步开放 | Lei 14.790/2023 |
| 新加坡 | 严格管控；远程赌博禁止（除豁免运营商） | Remote Gambling Act 2014 |

## 2. 虚拟货币棋牌的法律灰色地带

### "虚拟货币是否构成Prize"

这是全球棋牌游戏合规的核心法律问题。

#### 安全区（一般不构成赌博）

- 虚拟货币只能购买（单向流入），不能提现/转让
- 虚拟货币无真实市场价格（不可在第三方交易）
- 输赢不影响用户任何可提现资产
- 无代理/房卡等暗示线下结算的系统

#### 危险区（可能被认定为赌博）

- 虚拟货币可兑换为真金（直接提现）
- 存在第三方市场交易虚拟物品/货币
- "房卡"模式暗示线下金钱结算
- 排行榜奖励可提现的奖品
- 推荐系统含返佣/分润

## 3. 需要牌照的国家清单

### 明确需要赌博牌照才能运营（含棋牌类）

| 国家/地区 | 牌照类型 | 成本区间 | 审批时间 |
|-----------|----------|----------|----------|
| 英国 | UKGC Remote Operating Licence | GBP 1,000-100,000+/年 | 3-6个月 |
| 马耳他 | MGA B2C Licence (Type 1-4) | EUR 25,000-50,000 | 3-6个月 |
| 直布罗陀 | Gibraltar Gambling Licence | GBP 100,000+ | 6-12个月 |
| 曼岛 | Isle of Man GSC Licence | 按GGR比例 | 3-9个月 |
| 美国（部分州） | State Gaming Licence | 差异大，$10K-$500K+ | 3-18个月 |
| 澳大利亚 | 各州/领地牌照 | AUD 50,000+ | 6-12个月 |
| 库拉索 | Curaçao Gaming Licence（正在改革） | $20,000-$40,000 | 1-3个月 |
| 爱沙尼亚 | EMTA Gaming Licence | EUR 32,000+ | 3-6个月 |
| 菲律宾 | PAGCOR/CEZA Licence | $50,000-$200,000 | 2-6个月 |

### 特别注意：即使是"模拟赌博"也需要牌照的地区

- **比利时**：Belgian Gaming Commission将loot box认定为赌博，无牌照运营面临刑事起诉
- **荷兰**：KSA（Kansspelautoriteit）将部分loot box认定为赌博
- **澳大利亚**：模拟赌场类游戏必须R18+分级

## 4. 概率公示要求

### 全球概率公示法规

| 国家/地区 | 要求 | 法律依据 |
|-----------|------|----------|
| 中国 | 必须公示所有付费随机物品概率；每日抽取上限（单抽≤30次，十连≤3次，总计≤50次/日） | 文化部2016年通知 + NPPA后续规定 |
| 日本 | 必须公示gacha概率；complete gacha（需集齐多物品才能兑换）已禁止 | 景品表示法（JFTC指引） |
| 韩国 | 2025年修订后，所有随机付费物品必须公示概率，且概率须经第三方审计 | 游戏产业促进法修正案 |
| 比利时 | 禁止付费loot box（视为赌博） | 赌博法 |
| 荷兰 | 禁止或严格限制付费loot box | 博彩法（Wet op de kansspelen） |
| 欧盟（整体） | 无统一立法，但European Parliament多次呼吁监管 | 各成员国自行立法 |
| 美国 | 联邦层面无强制要求；部分州有提案 | 无统一联邦法 |
| 澳大利亚 | 含付费loot box必须至少M（15+）分级 | Classification Board新规 |

### Apple/Google平台层面要求

| 平台 | 要求 |
|------|------|
| Apple App Store | Guideline 3.1.1: 所有loot box/随机物品购买必须公示概率（全球） |
| Google Play | 建议公示但非强制（部分地区法规要求时必须） |

## 5. Loot Box全球监管趋势（2025-2026）

### 严格监管阵营（视为赌博或接近赌博）

- 比利时：实质禁止
- 荷兰：立法禁止中
- 澳大利亚：分级提高 + 立法讨论
- 西班牙：正在立法
- 英国：DCMS审查后建议加强监管，但未完全立法

### 中间立场（行业自律+部分监管）

- 日本：complete gacha禁止，但基本gacha允许
- 韩国：强制概率公示 + 第三方审计
- 中国：严格概率公示 + 数量限制

### 宽松/未立法阵营

- 美国联邦层面
- 多数东南亚国家
- 拉丁美洲多数国家
- 非洲多数国家
