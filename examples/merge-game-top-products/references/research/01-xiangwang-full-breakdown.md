# 01 "向往的生活"完整拆解

## 产品概况

- **开发/发行**：芒果TV官方出品（芒好玩平台），基于同名综艺IP
- **平台**：微信小游戏（核心渠道）、安卓/iOS
- **核心定位**：田园治愈系合成+模拟经营
- **目标用户**：女性为主，20-30岁，新一线城市占比高，综艺爱好者与模拟经营玩家交叉群体
- **IP基础**：《向往的生活》第八季总播放量突破15亿

## 核心循环：合成+家装+剧情三合一

### 合成层（Core）
- 采用"二合玩法"（Merge-2）作为核心机制
- 指尖合成，操作门槛极低，适配小游戏用户习惯
- 合成产出工具/材料 → 喂入家装层和经营层

### 家装/经营层（Meta）
- 还原综艺"蘑菇屋"场景
- 种田、植树、养鸡、养牛等农业活动
- 村庄建设：打造专属田园空间
- 后续版本解锁：萌宠、做菜、杂货铺、商店

### 剧情层（Narrative）
- 节目剧情和角色元素深度还原
- 与嘉宾的好感度系统和故事线
- 综艺IP的"慢生活"治愈基调贯穿始终

### 三层联动机制
```
合成产出工具/材料 → 推动家装/经营进度 → 解锁剧情节点
      ↑                                    |
      └──── 剧情奖励反馈新合成链 ←──────────┘
```

## 美术与调性

- 低饱和暖色调：米色、草绿、天蓝
- "无压力经营"的轻松氛围
- 与综艺IP"慢节奏"定位高度契合

## 市场表现

- 微信小游戏人气榜：空降第8名（2025.10.23），排名稳定
- 畅销榜：首日第85名，4天后升至第43名
- 吸金能力"中规中矩"，但作为赛道新品表现不俗

## 获量策略

- "短平快"低调买量策略
- 依靠IP影响力进行社交裂变传播
- 素材投放集中在10月，峰值达单日近1000条

## 版本更新与运营节奏

- 持续添加新功能模块（萌宠、做菜、杂货铺等）
- 依托综艺季播节奏做内容联动
- 小游戏平台更新灵活，可热更新

## 二代产品演进：《田园小镇》

- 新增种植-加工-贸易全链条玩法
- 取消倒计时套路（减少付费压迫感）
- 多任务同步推进机制

## 关键观察

1. 合成系统偏轻量化，核心在模拟经营而非深度合成树
2. IP加持降低获客成本，但也限制了海外拓展空间
3. "二合"机制降低操作门槛，适合小游戏平台的碎片化场景
4. 剧情层依赖IP内容供给，需要持续的综艺季播联动
