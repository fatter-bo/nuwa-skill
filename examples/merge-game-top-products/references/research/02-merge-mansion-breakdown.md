# 02 Merge Mansion拆解

## 产品概况

- **开发商**：Metacore Games（芬兰）
- **发行时间**：2020年
- **平台**：iOS / Android
- **合成机制**：Merge-2（合二为一）
- **游戏板**：非扩展固定板
- **累计收入**：$160M+（截至2025年）
- **全球畅销榜**：合成品类长期Top 2

## 核心设计：叙事驱动的合成

### 双循环结构：谜题+合成

```
叙事层：Boulton家族谜团 → 任务目标（修缮/清理）→ 推动剧情解锁
                ↕
合成层：物品生成 → 合成升级 → 产出工具 → 完成任务
```

- 每个完成的任务都推进故事，揭示Grandma Ursula的神秘过去
- 叙事不是装饰层，而是核心驱动力——玩家为了"知道接下来发生什么"而合成

### 合成机制细节

- Merge-2机制：两个相同物品即可合成
- 中央合成板（固定空间）+ 多个独立任务板
- 物品来源：生成器点击、任务奖励、宝箱掉落
- 合成树深度：8-12级（深层链需要数天完成）

### 叙事设计

- 主线：Maddie帮Grandma Ursula修缮豪宅
- 暗线：家族秘密逐步揭开
- 叙事节奏：每完成一组任务解锁一段剧情
- 角色系统：多个NPC各有故事线

## 运营与活动策略

### 活动事件体系

Metacore在2024-2025年大幅增加活动频率，活动数量几乎翻倍：

| 活动类型 | 机制 | 变现效果 |
|---------|------|---------|
| Merge It Up!（迷你Battle Pass） | 简化版BP，短周期，免费/付费双轨 | 最后一天收入飙升+138% |
| Secret Supply（小游戏） | 街机+合成混合，用神秘代币延长游戏 | 刺激硬通消耗，为BP购买铺垫 |
| Circus Caper（装饰活动） | 独立合成格，叙事+外观奖励 | 需30级才能激活，拉长生命周期 |
| PvE限时模式 | 扩展轮换，多实例重叠运行 | 持续参与感 |

### 收入驱动机制

- Battle Pass是最重要的收入驱动器
- "末日效应"：活动最后一天收入激增（+138%），源于"最后机会"购买心理
- Secret Supply活动通常安排在BP活动之前——先耗尽玩家硬通，再推BP付费
- 多个IAP选项在收入高峰期同时呈现

### 更新节奏

- 版本更新：约每2-4周一次大版本
- 新区域/房间持续解锁
- 活动轮转密度持续提升（2024.03-2025.03翻倍）

## 库存管理策略

- 固定板空间 → 自然产生空间压力
- 玩家需在"立即合成"vs"等待凑齐更优合成"之间决策
- 时间管理层：部分任务有完成计时器
- 三层策略：合成策略 + 库存管理 + 时间管理

## 付费设计

- 能量系统：限制会话时长，耗尽时自然触发付费
- 加速/跳过计时器
- Battle Pass双轨（免费/付费）
- 限时活动包
- 能量上限可能在长时间游戏中打断节奏

## 竞争优势总结

1. **叙事深度**：品类中最强的故事驱动力
2. **活动创新**：不断推出新活动类型，保持新鲜感
3. **收入策略**：精准的活动排期制造消费节奏
4. **品牌营销**：病毒式广告（Grandma系列广告片）
