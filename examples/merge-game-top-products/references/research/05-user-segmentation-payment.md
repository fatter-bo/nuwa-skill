# 05 用户分层与付费设计

## 用户分层模型

### 四层付费分层

| 层级 | 定义 | 占比（典型） | 贡献收入 | 行为特征 |
|------|------|-------------|---------|---------|
| 免费用户 | 从不付费 | 75-85% | 广告收入 | 每日1-2次短会话，能量自然恢复为主 |
| 小R（Minnow） | 月消费<$10 | 10-15% | 15-20% | 首充/月卡用户，偶尔看广告补能量 |
| 中R（Dolphin） | 月消费$10-$50 | 3-5% | 25-35% | 购买Battle Pass+事件包，活动活跃 |
| 大R（Whale） | 月消费>$50 | 1-2% | 40-50% | 购买高价包，硬通消耗大户，活动排名追求者 |

### 进阶细分（devtodev方法）

按收入降序排列：
- **Grand Whale**：Top 1% → 单人可贡献总收入5-10%
- **Whale**：Top 2-10%
- **Grand Dolphin**：Top 11-25%
- **Dolphin**：Top 26-50%
- **Minnow**：Top 51-100%（付费用户中）

## 各层级体验差异

### 免费用户体验
- 能量自然恢复限制每日2-4次会话（每次5-15分钟）
- 合成进度较慢，深层合成链需要数周
- 活动参与度受限：通常只能完成免费轨道60-70%
- 广告奖励是主要额外资源来源（4-6次/天激励视频）
- 核心体验完整但节奏显著慢于付费用户

### 小R体验（首充+月卡）
- 首充礼包打破"零付费"心理门槛
- 月卡提供每日稳定硬通收入 → 能量焦虑显著降低
- 合成进度提升约30-50%
- 可以在活动中达成中等目标
- 性价比最高的付费层级

### 中R体验（Battle Pass + 事件包）
- Battle Pass付费轨道提供独占奖励和加速
- 事件包在关键活动节点加速进度
- 合成进度提升约2-3倍
- 能量基本不再成为瓶颈
- 可以完成大部分活动目标

### 大R体验
- 购买高价包获取大量硬通和独占物品
- 活动排名靠前（排名本身成为消费动力）
- 合成进度接近最快速度
- 独占外观/装饰展示身份标识
- 客服优先/VIP体验（部分产品）

## 付费档位设计

### 首充漏斗

```
免费体验（D1-D3）
  → 赠送充裕硬货币（~100颗），体验付费级体验
  → 资源急剧稀缺（D4-D7），"先宠溺再变现"
  → 首充礼包弹出（$0.99-$4.99），极高性价比
  → 转化率：5-15%（品类平均）
```

### 月卡设计

| 参数 | 典型设计 |
|------|---------|
| 价格 | $3.99-$9.99 |
| 周期 | 30天 |
| 即时奖励 | 中量硬通 + 能量补满 |
| 每日奖励 | 少量硬通 + 能量 |
| 30天总价值 | 约等于$30-$50的散购硬通 |
| 留存绑定 | 必须每日登录领取 → 自然提升D30留存 |

### 付费金字塔

```
        ┌─────────┐
        │ $49-$99 │ ← 鲸鱼包（大量硬通+独占物品+高级生成器）
        ├─────────┤
        │ $19-$29 │ ← 高价值事件包（限时限量）
        ├─────────┤
        │ $9-$14  │ ← Battle Pass / 中端包
        ├─────────┤
        │ $3-$6   │ ← 月卡 / 首充包
        ├─────────┤
        │ $0.99-$2│ ← 冲动购买入门包
        └─────────┘
```

## 变现触发点设计

### 自然摩擦点 → 付费转化

| 摩擦点 | 出现时机 | 转化手段 | 目标层级 |
|--------|---------|---------|---------|
| 能量耗尽 | 每次会话末尾 | 能量包 / 激励视频 | 全层级 |
| 合成差一个 | 合成过程中 | 脉冲式购买（飘过屏幕） | 小R-中R |
| 活动即将结束 | 活动最后24h | 加速包 / 特价包 | 中R-大R |
| 空间不足 | 中期游戏 | 空间扩展包 | 小R-中R |
| Battle Pass未完成 | BP最后几天 | BP加速 / 直接购买等级 | 中R |
| 排名下滑 | 排名活动中 | 即时加速包 | 大R |

### 脉冲式购买 vs 弹窗Offer

- **脉冲式**（EverMerge模式）：物品在"差一个"时飘过屏幕，融入游戏流程
- **弹窗式**：直接弹出Offer窗口，打断游戏
- 行业趋势：脉冲比弹窗转化率更高，因为不产生打断感

## 广告变现层（混合模式）

### 激励视频触发点

| 场景 | 频次建议 | 目标层级 |
|------|---------|---------|
| 能量耗尽 | 4-6次/天 | 免费+小R |
| 奖励翻倍 | 随机出现 | 全层级 |
| 离线回归翻倍 | 每次回归1次 | 免费+小R |
| 减少等待 | 不限 | 免费+小R |
| 商店免费领取 | 每日1-2次 | 免费 |

**设计原则**：广告奖励是IAP的"试吃品"——让免费玩家体验付费的好处，培养付费意愿。

## 个性化定价趋势（2025-2026）

- 基于玩家行为的动态定价
- 不同细分用户看到不同的商店和Offer
- A/B测试驱动的持续优化
- 微细分可带来20%+的ARPU提升
- "保底系统"（Pity System）：保证N次付费后必获高价值奖励
