# 04 运营节奏分析

## 版本更新频率对比

| 产品 | 大版本周期 | 热更/小版本 | 核心内容节奏 |
|------|-----------|-----------|-------------|
| Merge Mansion | 2-4周 | 每周 | 新区域/房间持续解锁 |
| EverMerge | 2-3周 | 每周（周二/周六） | 新角色/新世界岛屿 |
| 向往的生活 | 1-2周（小游戏热更） | 灵活 | 新功能模块逐步开放 |

## 限时活动轮转

### Merge Mansion活动密度演变
- 2024.03-2025.03期间活动数量几乎翻倍
- 新活动类型密集推出（Merge It Up!、Secret Supply、Circus Caper等）
- 多活动实例重叠运行，玩家始终有活动可参与

### EverMerge活动体系
- **9种活动类型**，品类最多
- 季节通行证（Season Pass）机制
- 主要活动每周二/周六刷新
- 每四周用特殊活动替换常规活动，保持新鲜感
- 节日活动（圣诞/春节）覆盖常规排期

### 活动类型矩阵

## 付费活动模式

### 活动排期的收入策略（Merge Mansion案例）

```
Week 1: Secret Supply（硬通消耗活动）
  → 耗尽玩家硬通货币库存
Week 2: Merge It Up!（Battle Pass活动）
  → 硬通不足 → 触发IAP购买
  → 最后一天收入飙升+138%（"末日效应"）
```

这种"先耗后卖"的活动排期是精心设计的收入最大化策略。

### Battle Pass收入驱动

- Battle Pass是Merge Mansion最重要的收入驱动器
- 收入增长从2024.11到2025.03持续可见
- 免费轨道保持非付费玩家活跃
- 付费轨道提供高价值独占奖励
- 双轨设计：非付费玩家也能体验活动，降低排斥感

### "末日效应"（Last-Day Surge）

活动最后一天的收入飙升现象：
- Merge It Up!最后一天：+138%收入
- 心理机制："最后机会"紧迫感 + 沉没成本（已投入时间）
- 设计启示：活动倒计时要醒目，最后24小时推送特价包

## 2025-2026行业趋势

### 活动模板化（Templatisation）
- 团队更精简、成本更敏感
- 活动框架标准化，内容换皮高效产出
- 每个赛季"感觉新鲜"但底层系统复用

### 个性化（Personalisation）
- 微细分用户群体，精准推送活动难度和奖励
- 实时A/B测试调优活动参数
- 不同用户看到不同的活动商店和定价

### AI辅助LiveOps
- AI生成活动文案/美术素材
- 自动化活动参数调优
- 预测性分析优化活动排期

### 活动可实时调整
- 大型单体更新持续减少
- LiveOps团队期望在活动运行中实时调整内容
- 动态难度/奖励调整
