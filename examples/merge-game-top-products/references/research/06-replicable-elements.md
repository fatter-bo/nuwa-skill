# 06 可复制要素提取

## 分类框架：品类通用 vs 产品独有

### 品类通用要素（可直接复用）

这些是合成游戏品类的"基础设施"，缺少任何一项都会导致产品竞争力不足：

#### 1. 合成即时反馈系统
- 每次合成必须产出视听正反馈（动画+音效+粒子效果）
- 15-30秒触发一次奖励反馈循环
- "完成效应"（Completion Effect）：将混乱整理为秩序的心理满足

#### 2. Meta层叠加
- 纯合成游戏留存显著低于有Meta层的产品——这是被数据验证的品类铁律
- Meta层类型可选（修缮/建造/收集/探索），但必须存在
- Meta层决定产品天花板

#### 3. 能量系统
- 会话节奏控制（5-20分钟/次）
- 变现触发点（能量耗尽=最自然的付费时机）
- 内容消耗节流阀
- 约占休闲游戏收入的三分之一

#### 4. 双货币体系
- 软货币（游戏内产出，基础消耗）+ 硬货币（稀缺，核心变现）
- "先宠溺再变现"：新手阶段大量赠送硬通

#### 5. 限时活动系统
- Battle Pass/Season Pass双轨设计
- 活动轮转保持新鲜感
- "末日效应"在活动最后一天制造收入脉冲

#### 6. 空间管理压力
- 有限空间创造策略深度
- 空间不足是自然的变现触发点

#### 7. 生成器系统
- 生成器是合成链的物品来源
- 冷却/能量消耗控制产出节奏
- 产出的适度随机性维持新鲜感

### 产品独有要素（不可直接复用，需要独特创造）

这些是各头部产品的差异化护城河：

#### Merge Mansion独有
- **家族悬疑叙事**：Boulton家族故事是独一无二的IP资产
- **Grandma病毒式广告**：创意营销建立的品牌认知
- **"先耗后卖"活动排期**：Secret Supply→Battle Pass的精准收入节奏
- **固定板+多任务板**的独特空间设计

#### EverMerge独有
- **经典童话IP融合**：灰姑娘/穿靴子的猫等角色
- **9种活动类型**：品类最丰富的活动矩阵
- **三系统平衡**（能量/资源/库存）：相互依赖的复杂平衡
- **5合2奖励**：20%进度提升的掌握感设计

#### 向往的生活独有
- **综艺IP还原**：蘑菇屋场景、嘉宾角色、好感度系统
- **微信社交裂变**：平台级社交分发能力
- **田园治愈调性**：低饱和暖色调的独特美术方向
- **综艺联播联动**：季播内容持续注入新素材

#### Merge Dragons!独有
- **"发现"设计哲学**：刻意隐藏完整合成链，保持探索神秘
- **关卡+营地双模式**：品类独创的双板结构
- **治愈土地机制**：独特的地图解锁方式

## 新产品设计的"必做-应做-可做"清单

### 必做（P0）——没有这些不要上线
- [ ] 合成即时反馈系统（视听正反馈）
- [ ] 至少一种Meta层（建造/修缮/收集）
- [ ] 能量系统（控制会话节奏）
- [ ] 双货币体系（软通+硬通）
- [ ] 首充礼包 + 月卡
- [ ] 每日签到系统

### 应做（P1）——上线3个月内补齐
- [ ] Battle Pass / Season Pass
- [ ] 3-4种限时活动类型
- [ ] 活动排期策略（"先耗后卖"）
- [ ] 用户分层差异化体验
- [ ] 回流机制
- [ ] 推送策略

### 可做（P2）——根据产品方向选择
- [ ] 深度叙事系统
- [ ] 社交系统（互赠/排行/合作）
- [ ] IP联动
- [ ] 装饰/外观系统
- [ ] PvE/PvP竞技活动
- [ ] 个性化定价

## 品类成功公式

```
合成游戏成功 = 即时满足感(合成反馈) × 中期驱动(Meta层深度) × 长期绑定(活动+社交+收集)
                × 变现效率(摩擦点→付费转化)
                × 获量效率(CPI < LTV)
```

**核心洞察**：合成机制是品类入场券（门槛低、差异小），Meta层和运营才是真正的竞争维度。
