## 2. 常见迁移路径

### 路径1: 中国风 → 日系清新

**色彩调整**：
- 提高整体饱和度20-30%
- 色温从暖色偏移向中性/冷色
- 增加粉色、浅蓝、薄荷绿等明亮色
- 降低黑色使用量，阴影改用深蓝/深紫

**线条调整**：
- 水墨飘逸线条 → 均匀粗细的干净描边
- 增加黑色描边或深色同色系描边
- 线条端点从渐隐 → 圆头/方头

**光影调整**：
- 水墨渐变 → 赛璐璐分阶（2-3个明度层级）
- 阴影边缘从模糊 → 锐利

**提示词差异**：
```
# 中国风原版
chinese ink painting, traditional, flowing brushwork, rice paper texture,
muted colors, elegant, xianxia style

# 日系清新目标
anime style, cel-shaded, clean lineart, bright colors,
soft pastel palette, cute, sparkle effects, anime screenshot coloring
```

### 路径2: 中国风 → 韩系Q版

**色彩调整**：
- 饱和度微降，加入粉彩（pastel）色调
- 整体偏暖但柔和
- 增加马卡龙配色元素

**线条调整**：
- 线条变圆润
- 描边变细且均匀
- 增加可爱的装饰细节

**比例调整**：
- 从标准头身比 → 2.5-3.5头身
- 头部放大、眼睛放大
- 手脚缩小但保留细节

**提示词差异**：
```
# 韩系Q版目标
chibi style, cute character, pastel colors, soft shading,
rounded features, big eyes, small body, kawaii,
MapleStory style, Korean mobile game art
```

### 路径3: 欧美写实 → 日系动漫

**色彩调整**：
- 大幅提高饱和度
- 减少灰色/棕色的使用
- 皮肤色从自然肤色 → 偏白/偏粉
- 头发颜色可更鲜艳（非自然色OK）

**线条调整**：
- 添加明确的黑色/深色描边
- 线条均匀化
- 细节简化但轮廓更清晰

**光影调整**：
- PBR写实光照 → 赛璐璐2-3阶
- 去除次表面散射等写实效果
- 高光从渐变 → 锐利白块

**比例调整**：
- 眼睛放大
- 鼻子嘴巴缩小/简化
- 下巴变尖

**提示词差异**：
```
# 欧美写实原版
realistic, photorealistic, detailed skin texture, natural lighting,
volumetric, subsurface scattering, 8k UHD

# 日系动漫目标
anime style, cel shading, clean lineart, vibrant colors,
anime eyes, simplified features, 2D illustration,
anime screenshot, flat color with sharp shadows
```

## 3. 文化注意事项

### 符号与图案
| 文化元素 | 中国 | 日本 | 韩国 | 注意事项 |
|----------|------|------|------|---------|
| 云纹 | 祥云（圆润多层） | 松形云（扁平简洁） | 类日系 | 形状差异明显 |
| 花卉 | 牡丹/梅花/莲花 | 樱花/菊花 | 木槿/芍药 | 国花选择很重要 |
| 龙 | 五爪龙、东方龙 | 四爪或三爪 | 四爪 | 爪数有文化含义 |
| 服饰 | 汉服/旗袍 | 和服/巫女服 | 韩服 | 领口/腰带/裙型完全不同 |
| 建筑 | 斗拱飞檐/红墙黄瓦 | 鸟居/唐破风 | 丹青/月台 | 屋顶曲线和装饰差异大 |

### 禁忌事项
- 不同文化的颜色含义不同（如白色在东亚可代表丧葬）
- 宗教符号使用需谨慎（万字符、十字架等）
- 某些手势在不同文化中含义不同
- 历史敏感题材的处理需要慎重

## 4. 技术实现参考

### 相关研究
- **CartoonGAN (CVPR 2018)**：Tsinghua University，专注于照片转动漫风格
- **U-GAT-IT (2020)**：NCSOFT资助，用于游戏角色设计pipeline
- **StyleGAN-NADA (2021)**：通过CLIP文本引导将预训练模型迁移到目标域（如"anime drawings"），无需配对训练数据
- **NAVER Webtoon Interactive Cartoonization (CVPR 2023)**：允许艺术家控制输出细节程度
- **InstantStyle (2024)**：风格-内容解耦的通用风格迁移
- 来源：[SyncedReview - CartoonGAN](https://syncedreview.com/2018/07/14/reproducing-japanese-anime-styles-with-cartoongan-ai/) | 可信度：极高（学术综述）

### CivitAI风格模型参考
- **Mistoon_Anime**：明亮2D动漫风格，色彩鲜艳
- **Illustrij**：2.5D风格，平涂动漫+微妙3D光影
- 使用LoRA可以进一步微调到特定子风格
- 来源：[CivitAI](https://civitai.com/) | 可信度：高（社区验证模型）
