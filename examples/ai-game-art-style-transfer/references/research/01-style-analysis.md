# 维度1：风格分析方法论

## 研究目标
如何量化分析一套游戏美术的风格特征，建立可执行的风格拆解框架。

---

## 1. 色彩体系分析

### HSL色彩空间量化
- **Hue（色相，0-360）**：红(0)、黄(60)、绿(120)、青(180)、蓝(240)、品红(300)
- **Saturation（饱和度，0-100）**：数值越高颜色越鲜艳
- **Lightness（明度，0-100）**：决定颜色趋向白还是黑
- 来源：[Pav Creations - Color Theory for Game Art](https://pavcreations.com/color-theory-for-game-art-design-the-basics/) | 可信度：高（一手教程）

### 色板结构方法
- 调色板由多个"色阶（ramp）"组成，每个ramp是一组相邻色相的颜色（如红色ramp、绿色ramp、棕色ramp）
- 游戏美术分析时应提取：**主色调**（画面占比>40%）、**辅助色**（20-40%）、**强调色**（<10%，用于交互/高亮）
- 参考Gurney方法：选择几个关键色并通过混合生成其余色板，相当于从色彩空间中切出一个子体积
- 来源：[OpenGameArt - Color Palettes](https://opengameart.org/content/chapter-5-color-palettes) | 可信度：高（一手资源）

### 工具方法
- 使用Adobe Color / Coolors从素材截图中提取主色板
- 用Python的PIL/OpenCV做HSL直方图统计，量化色相分布
- 对比原版与目标风格的HSL分布差异

## 2. 线条特征分析

### 关键维度
- **粗细**：像素宽度（1px极细线→5px+粗描边）
- **圆角/锐角**：圆润风格 vs 硬朗风格
- **有无描边**：日系动漫常用黑色描边，欧美写实常无描边
- **线条颜色**：纯黑、深色同色系、彩色描边各有不同风格感受

### 量化方法
- 使用边缘检测（Canny/Sobel）提取素材线稿
- 测量平均线条宽度和曲率分布
- 统计描边颜色的HSL值

## 3. 光影风格分类

### 五级分类体系
| 级别 | 名称 | 特征 | 典型游戏 |
|------|------|------|---------|
| L1 | 纯平涂 | 无明暗变化，纯色块 | 部分超休闲游戏 |
| L2 | 赛璐璐 | 2-3级明暗阶梯，锐利阴影边缘 | 日系动漫风格游戏 |
| L3 | 软赛璐璐 | 阴影边缘有少量过渡，2.5D感 | 韩系手游 |
| L4 | 半写实 | 渐变光影但保留手绘感 | 《原神》等 |
| L5 | 写实渲染 | 完整PBR光照模型 | AAA写实游戏 |

- 来源：[Cel Shading Wikipedia](https://en.wikipedia.org/wiki/Cel_shading)、[GarageFarm Cel Shading Guide](https://garagefarm.net/blog/cel-shading-a-comprehensive-guide) | 可信度：高

## 4. 比例规范

### 头身比分类
| 类型 | 头身比 | 风格倾向 |
|------|--------|---------|
| 极Q版 | 2-3头身 | 可爱/卡通 |
| Q版 | 3-5头身 | 轻松/休闲 |
| 正常 | 5-7头身 | 日系标准 |
| 写实 | 7-8头身 | 写实/欧美 |
| 夸张 | 8+头身 | 英雄/超人体型 |

### 分析要点
- 测量角色素材的实际头身比
- 记录头部占比、四肢比例、手脚大小
- Q版角色通常手脚变小、眼睛放大

---

## 5. 纹理质感分析

### 质感分类
- **干净/数字感**：无纹理噪点，纯色或渐变
- **纸质/手绘感**：叠加纸张纹理或笔触噪点
- **像素感**：低分辨率像素风格
- **厚涂/油画感**：可见笔触和颜料堆叠
- **水彩/水墨感**：晕染边缘、透明叠加
