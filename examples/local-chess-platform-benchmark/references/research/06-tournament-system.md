# 维度六：竞技/赛事系统

## 1. JJ竞技的赛事体系

### 赛制类型

JJ比赛提供多种赛制，覆盖不同竞技深度：

**定时赛（Schedule Tournament）**
- 在固定时间开赛，报名后等待开赛
- 所有选手同时开始，限时N局
- 按总积分排名，前N名获得奖品
- 适用场景：日常赛事、节假日特别赛

**打立出局赛（Knockout/Sit-and-Go）**
- 适用于预赛和资格赛阶段
- 150+人参赛，分数降到0或以下立即淘汰
- 淘汰至剩余目标人数后按排名决定名次
- 高强度、强对抗，一局定生死

**积分赛（Points League）**
- 长周期赛事（周赛/月赛/赛季）
- 每场比赛积累积分
- 赛季末按总积分排名
- 适合培养忠实竞技用户

**复式团体赛**
- JJ斗地主冠军杯核心赛制
- 4-6人组成战队
- 两队分别在两桌打一样的牌（复式）
- 消除运气因素，突出技术含量

**双人赛（Double）**
- 双骄杯赛制，两人搭档参赛
- 配合默契度成为竞争维度

**混团赛**
- 要求战队至少包含2名女性选手
- 比赛全程2名女选手必须同时上场
- 推动多元化参与

## 2. 途游TUPT赛事体系

### 赛事品牌
TUPT（途游扑克锦标赛）是途游自主孵化的赛事IP：
- 线上日赛：每天5000余场
- 累计参赛人数：近800万
- 累计关注人次：近1亿

来源: [TUPT赛事规模](https://sports.sina.cn/others/qipai/2017-04-27/detail-ifyetwsm0777464.d.html)

### 赛事结构
```
线上海选（每日5000+场）
    ↓
四地分站赛（武汉/哈尔滨/西安/深圳）
    ↓
总决赛
```

### 电竞化元素
- OB系统（实时观战）
- 直播（多平台同步）
- 专业解说
- 线下专业赛程流程

来源: [途游棋牌电竞化](https://www.jiemian.com/article/1735001.html)
来源: [TUPT总决赛](https://sports.sina.cn/others/qipai/2017-11-26/detail-ifypathz6158426.d.html)

### 赛事合伙人模式
- 全国有能力承办棋牌比赛的团体/组织/个人均可成为合伙人
- 裂变式扩张赛事覆盖范围
- 降低线下赛事运营成本

## 3. 微乐的轻竞技模式

微乐的赛事体系相对轻量，以社交娱乐为主：
- **好友对局**：核心场景是熟人房
- **匹配竞技**：在线匹配的段位赛/排位赛
- **小型赛事**：平台内小规模赛事活动
- **无重度电竞**：不做职业化赛事，保持休闲社交定位

## 4. 合规边界分析

### 棋牌赛事的法律红线

**合法区间**
- 以虚拟物品为媒介的竞技比赛
- 奖品为实物（家电、手机等），非现金
- 参赛费用合理，非高额投注
- 赛事有明确规则和公平机制
- 获得体育管理部门认可（途游获国家体育总局棋牌管理中心授牌）

**风险区间**
- 虚拟货币可逆向兑换为人民币→涉嫌赌博
- "金商"体系（第三方代充/代兑）→灰色地带
- 高频次、高金额的竞赛→可能被认定为变相赌博
- 用户投诉"后台操控"→信任危机

来源: [JJ斗地主用户投诉](https://finance.sina.cn/chanjing/gsxw/2021-09-27/detail-iktzqtyt8354611.d.html)
来源: [竞技世界服务协议](https://www.jj.cn/agreement/regService.html)

### 合规设计要点

| 要点 | 说明 |
|------|------|
| 虚拟货币单向性 | 人民币→虚拟货币可以，反向不可以 |
| 奖品实物化 | 赛事奖品为实物/服务，避免现金等价物 |
| 参赛门槛合理 | 报名费不应过高，避免"赌注"嫌疑 |
| 概率透明 | 涉及随机元素需公示概率 |
| 防沉迷 | 限制未成年人参赛，设置消费限制 |
| 反作弊 | IP检测、设备指纹、行为分析 |
| 体育总局背书 | 与官方体育机构合作增加合法性 |

### 三平台合规策略对比

| 平台 | 合规策略 | 风险点 |
|------|---------|--------|
| 微乐 | 轻竞技+社交，无重度赛事 | 风险最低 |
| 途游 | 体育总局合作，TUPT品牌赛事 | 中等，依赖官方背书 |
| JJ竞技 | 专业竞技定位，严格资格审核 | 较高，"金商"和投诉风险 |
