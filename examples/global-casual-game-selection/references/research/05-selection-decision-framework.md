# 维度五：选品决策框架

## 综合评分模型

### 五维评分体系

对每个"品类×市场"组合进行1-10分评估：

```
选品得分 = 市场空白度(W1) × 品类热度(W2) × 开发复杂度反向(W3) × AI可替代度(W4) × 变现潜力(W5)

建议权重：W1=0.25, W2=0.20, W3=0.20, W4=0.15, W5=0.20
```

### 各维度评分标准

#### 1. 市场空白度（0.25权重）
| 分数 | 定义 |
|------|------|
| 9-10 | 目标市场Top 200无同品类产品，有搜索需求 |
| 7-8 | Top 200中同品类<5款，无头部产品 |
| 5-6 | 同品类10-20款，但无$1M+月收入产品 |
| 3-4 | 同品类20-50款，有明确头部 |
| 1-2 | 同品类50+款，高度饱和 |

#### 2. 品类热度（0.20权重）
| 分数 | 定义 |
|------|------|
| 9-10 | 全球YoY增长>20%，多市场同时爆发 |
| 7-8 | 全球YoY增长10-20%，至少2个主要市场增长 |
| 5-6 | 全球持平或微增，个别市场有增长 |
| 3-4 | 全球下降但个别市场仍有机会 |
| 1-2 | 全球持续下降，无增长信号 |

#### 3. 开发复杂度反向（0.20权重）
| 分数 | 定义 |
|------|------|
| 9-10 | 2D简单机制，1-2人团队可完成，<$20K |
| 7-8 | 2D中等复杂度，3-5人团队，$20-50K |
| 5-6 | 2D/3D混合，5-10人团队，$50-150K |
| 3-4 | 3D或复杂系统，10-20人团队，$150-500K |
| 1-2 | AAA级别，20+人团队，>$500K |

#### 4. AI可替代度（0.15权重）
| 分数 | 定义 |
|------|------|
| 9-10 | 美术/音效/关卡设计均可AI大幅辅助，成本降70%+ |
| 7-8 | 主要美术资产可AI生成，人工微调即可 |
| 5-6 | 部分环节可AI辅助，核心创意仍需人工 |
| 3-4 | AI仅辅助边缘环节，核心需大量人工 |
| 1-2 | AI几乎无法替代，需全人工制作 |

#### 5. 变现潜力（0.20权重）
| 分数 | 定义 |
|------|------|
| 9-10 | 品类ARPDAU>$0.30，变现模型成熟且多元 |
| 7-8 | ARPDAU $0.15-0.30，IAP+IAA混合模式验证 |
| 5-6 | ARPDAU $0.08-0.15，主要依赖广告 |
| 3-4 | ARPDAU $0.03-0.08，变现模式单一 |
| 1-2 | ARPDAU<$0.03，变现模型未验证 |

## 决策门槛与红线规则

### 必要条件（任一不满足则放弃）
1. 市场空白度 ≥ 4（不进入高度饱和市场）
2. 品类热度 ≥ 3（不做持续萎缩品类）
3. 加权总分 ≥ 6.0

### 红线规则
- 首次出海团队不做Match-3（成功率1.4%）
- 预算<$50K不做3D或重度LiveOps品类
- 无本地化团队不做文化敏感市场（日本/中东）
- 不同时进入3个以上市场（资源分散）
