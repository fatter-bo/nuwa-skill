# 维度二：品类渗透率差异分析

## 全球休闲游戏品类格局（2025数据）

### 下载量排名
1. 模拟类（Simulation）：~9.8B年下载
2. 解谜类（Puzzle）：~9.7B年下载
3. 街机类（Arcade）：~9.6B年下载

### 收入排名（IAP）
1. 策略类：领先
2. 解谜类：$10B+（YoY +14%）
3. 射击类：第三

### Puzzle子品类收入结构
- Match-3/Merge/Blast三大子品类占Puzzle总收入70-75%
- Match-3：$2.7B收入（+7% YoY），但安装量下降17%至3.94亿
- Merge：贡献Puzzle品类60%的YoY增长
- 新兴子品类贡献剩余40%增长：Sort（13%）、Block（13%）

### "A国火爆B国空白"的典型品类机会

1. **Merge品类 → 东南亚/中东**：欧美日已验证，东南亚和中东渗透率极低
2. **Sort/Block新兴品类 → 全球新兴市场**：Magic Sort、Color Block Jam在欧美爆发，但亚洲和中东几乎空白
3. **棋牌变种 → 跨区域移植**：东南亚地方棋牌（如印尼Domino）在本地火爆，可参考机制做全球化包装
4. **模拟经营 → 东南亚**：该市场轻度游戏需求正在上升，开发成本可控

## 识别品类机会的方法论

### 三步筛选法
1. **榜单扫描**：对比目标国家Top 200免费/畅销榜的品类分布，找出缺席的热门品类
2. **渗透率计算**：目标品类在该国下载量 / 该国总游戏下载量，低于全球均值即为潜在机会
3. **需求验证**：检查该国是否有该品类的搜索热度（Google Trends/App Store搜索建议），有搜索无产品 = 空白市场确认

### 关键数据信号
- 品类下载量下降但收入持平/上升 → 成熟市场，新手难进
- 品类下载量上升但竞品少 → 蓝海窗口期
- 头部产品ARPDAU异常高 → 变现模型已验证，可跟进
