# 全球休闲游戏选品决策系统

> 选品的本质不是"什么游戏好做"，而是"什么品类在什么市场存在供需错配"。找到一个A国已验证但B国空白的品类，比在红海市场做微创新的成功概率高一个数量级。

## 使用规则

| 用户说 | 触发动作 |
|--------|----------|
| 「该做什么游戏」 | 输出完整选品分析（品类扫描→市场筛选→竞品评估→综合评分→推荐方案） |
| 「分析XX市场」 | 输出目标市场特征画像 + 品类机会矩阵 + 进入策略建议 |

## 一、榜单分析方法论

### 1.1 核心工具矩阵

| 工具 | 强项 | 适用场景 | 价格层级 |
|------|------|---------|---------|
| Sensor Tower（含data.ai） | 全球下载/收入估算、Game IQ分类（70+子品类） | 品类规模评估、跨国对比 | 企业级 |
| AppMagic | 西方Tier-1休闲市场分析 | 休闲品类深度报告 | 中等 |
| GameRefinery | 功能级竞品分析 | 竞品Feature对标 | 企业级 |
| Naavik | 深度行业分析 | 品类趋势判断 | 订阅制 |

### 1.2 App Store/Google Play榜单算法

**免费榜/付费榜**：核心驱动因素是**下载速度**（非累计量），使用4-7天移动平均值。

**畅销榜**：核心驱动因素是**收入速度**（IAP收入），与下载量无直接关系。

**次要因素**：评分与评论量、留存率（卸载率反向影响）、关键词相关性（Google Play权重更高）。

**关键认知**：
- 同一排名在不同国家代表完全不同的绝对值（美国Top 100 vs 越南Top 100差距可达100倍）
- 榜单存在"惯性效应"——高排名带来自然流量，进一步巩固排名
- 周末冲高/工作日回落 → 依赖买量；排名平稳 → 自然流量健康

### 1.3 Sensor Tower Game IQ分类体系

Puzzle品类已细分为19个子品类：
- **头部三强**：Swap（Match-3）、Blast、Merge — 合占品类收入70-75%
- **新兴增长**：Sort（排序）、Block（方块）、Match Pair（配对）
- 支持按子品类、题材、美术风格、变现机制等维度交叉分析

### 2.3 品类空白识别三步法

```
Step 1: 榜单扫描 → 对比目标国Top 200品类分布，找缺席的全球热门品类
Step 2: 渗透率计算 → 品类下载量 / 该国总游戏下载量，低于全球均值 = 潜在机会
Step 3: 需求验证 → 检查Google Trends/App Store搜索建议，有搜索无产品 = 空白确认
```

### 4.2 五类竞争壁垒

1. **内容壁垒**：头部Match-3有数千关卡积累，Merge有深度合成树+持续事件
2. **品牌壁垒**：Candy Crush等自带搜索流量，ASO护城河深
3. **数据壁垒**：长期用户行为数据驱动精准LiveOps和LTV优化
4. **资金壁垒**：成功休闲游戏需$100万+开发+推广预算
5. **LiveOps壁垒**：社交功能使留存提升30-40%，形成玩家习惯锁定

### 4.3 竞品密度五步评估法

```
Step 1: 竞品计数 → 目标市场搜索品类关键词，统计Top 200竞品数
Step 2: 头部集中度 → Top 5占品类总收入比（>70%为高集中）
Step 3: 新品存活率 → 6个月内新品仍在Top 500的比例
Step 4: CPI基准 → 目标品类在目标市场的平均CPI（>$3为高竞争）
Step 5: 创意饱和度 → 月广告素材产出量（参考线：RPG平均251条/月）
```

## 八、决策启发式（十条黄金法则）

### 1. "搬砖优于发明"
在A国Top 100验证过的品类搬到B国空白市场，成功率远高于在红海市场做微创新。优先找供需错配，而非做第N+1个Match-3。

### 2. "下载跌但收入稳 = 别进"
品类下载量下降但收入持平或上升，说明存量用户在付费但新增已枯竭——这是成熟市场的信号，新手极难分一杯羹。

### 3. "1.4%法则"
Match-3在2025年H1的新品成功率仅1.4%。在进入任何品类前，先查该品类过去6个月新品突破$10万月收入的比例。低于5%的品类需要三倍于行业平均的差异化才值得尝试。

### 4. "CPI是市场温度计"
目标品类在目标市场的CPI > $3 → 红灯；$1-3 → 黄灯；< $1 → 绿灯。CPI直接决定你的ROI天花板。

### 5. "社交渠道用户值两倍"
微信小游戏中社交渠道用户留存率高20%，获取成本为零。任何选品方案都应优先评估"这个品类能否激发分享欲"。

### 6. "沙特ARPU是中国的5倍"
中东不是低价市场。沙特/阿联酋的高付费能力常被忽视，但需要文化适配投入。高ARPU市场值得额外的本地化成本。

### 7. "先小游戏验证，再全球铺开"
微信小游戏的2026新政（首1000万流水零分成）使其成为理想的品类验证平台。在微信上跑通核心循环和变现模型，再考虑出海。

### 8. "AI杠杆优先选2D"
AI工具对2D休闲游戏的效率提升最大（成本降70%+）。3D或需要大量手绘原创美术的品类，AI的杠杆效应会大幅缩水。选品时，同等条件下优先选择AI可深度介入的品类。

### 9. "窗口期以月计"
Sort/Block等新兴品类的蓝海窗口期通常只有6-12个月。当你看到数据信号时，竞争者也看到了。决策速度本身就是竞争优势。

### 10. "不同时打三个市场"
资源分散是独立团队的第一大杀手。选定一个"品类×市场"组合，做到Top 100，再考虑第二个市场。宁可在一个市场做到极致，不可在三个市场都做到平庸。
