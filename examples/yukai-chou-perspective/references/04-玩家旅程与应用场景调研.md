# 04 - 玩家旅程与应用场景调研

## 调研时间：2026-04-12

## 四阶段玩家旅程（Level 2 Octalysis）

### Phase 1: Discovery（发现）
- 覆盖：用户听说产品到注册之前的一切
- 关键驱动力：
  - CD1 Epic Meaning（"这个产品在做一件很酷的事"）
  - CD7 Unpredictability（"这看起来很有趣，我想试试"）
  - CD5 Social Influence（"朋友推荐"/"名人背书"）
- 对应指标：Acquisition（获客）

### Phase 2: Onboarding（引导）
- 覆盖：注册后到掌握基本技能
- 关键驱动力：
  - CD2 Accomplishment（"我学会了！"第一次胜利）
  - CD3 Creativity（让用户尽早体验核心玩法循环）
  - CD8 Loss Avoidance（"教程奖励即将过期"）
- 对应指标：Activation（激活）
- 结束标志：用户达成"第一个重大胜利"（First Major Win-State）

### Phase 3: Scaffolding（脚手架）
- 覆盖：用户形成日常回访习惯
- 关键驱动力：
  - CD4 Ownership（"我的角色/账户/收藏"）
  - CD5 Social（"公会/好友/竞争"）
  - CD6 Scarcity（"每日限量/体力系统"）
- 对应指标：Retention（留存）
- 特征：大多重复性操作，需要足够多的驱动力维持

### Phase 4: Endgame（终局）
- 覆盖：老玩家觉得"都玩过了"
- 关键驱动力：
  - CD1 Epic Meaning（"我是这个社区的领袖"）
  - CD3 Creativity（"棋类/策略类永不重复"——终局圣杯）
  - CD7 Unpredictability（"随机事件/彩蛋/动态挑战"）
- 对应指标：Loyalty + Monetization
- Chou核心观点："真正的游戏从你满级之后才开始"
- 失败案例：大多数产品完全不做Endgame设计

### Endgame关键技巧
- **Collection Sets + Variable Rewards**：未完成的收藏+稀有掉落
- **The Big Burn**：极其昂贵的身份象征道具
- **Endless Challenges**：算法动态难度，永无止境
- **Recursive Mechanics**：老玩家塑造新玩家体验
- **Moats**：随机组合挑战迫使老手用新方式思考

## 休闲游戏与棋牌应用映射

### 休闲游戏（合成/消除/模拟经营）
| 驱动力 | 表现形式 |
|--------|----------|
| CD2 成就 | 关卡星级、里程碑奖励、赛季排名 |
| CD3 创意 | 合成路径选择、基地装饰自由度 |
| CD4 所有权 | 收集图鉴、角色养成、虚拟货币 |
| CD6 稀缺 | 体力系统、限时活动、稀有道具 |
| CD7 不可预测 | 抽卡/扭蛋、随机事件、宝箱 |
| CD8 损失规避 | 连续签到断裂惩罚、排名衰减 |

### 棋牌游戏（德扑/斗地主/麻将）
| 驱动力 | 表现形式 |
|--------|----------|
| CD2 成就 | 段位系统、赛季段位重置、锦标赛 |
| CD3 创意 | 牌局策略空间、bluff技巧、概率计算 |
| CD4 所有权 | 欢乐豆/筹码积累、个人战绩、头像框 |
| CD5 社交 | 好友房、公会赛、实时聊天、送礼 |
| CD6 稀缺 | 高额桌入场门槛、VIP特权 |
| CD7 不可预测 | 发牌随机性（天然自带）、对手行为不可控 |
| CD8 损失规避 | 下注后不想弃牌（沉没成本）、段位掉落恐惧 |

## Zynga失败案例分析（来自White Hat文章）
- Zynga精通黑帽技巧，"数据驱动设计"创造了强劲短期指标
- 股价在几个月内从$14.69暴跌至$3以下
- 原因："when people don't feel good playing, they will leave when possible"
- 唯二成功产品：Zynga Poker和Words with Friends——因为底层是Poker和Scrabble这样的永恒游戏，意外保留了CD3（创意/反馈）
- **教训**：纯黑帽留存不可持续，需要白帽基础
