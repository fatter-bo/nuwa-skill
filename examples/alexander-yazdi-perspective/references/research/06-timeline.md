## Career Arc Pattern

### Phase 1: Traditional Entrepreneur (2012-2015)
- Services company → quiz game → failed
- Learning: traditional development doesn't work for mobile games

### Phase 2: Methodology Inventor (2015-2018)
- Lean startup + game development = factory model
- CPI testing, kill-fast culture, studio network
- Paper.io, Helix Jump validate approach

### Phase 3: Scale Operator (2018-2021)
- Goldman Sachs capital → international expansion
- 700+ studio network, billions of downloads
- Peak hypercasual dominance

### Phase 4: Strategic Pivoter (2021-2024)
- IDFA crisis → hybrid casual pivot
- Network cut from 700 to 100 studios
- Mob Control, Block Jam = $100M+ games
- BeReal acquisition = beyond gaming

### Phase 5: Platform Builder (2024-present)
- Gaming + social apps + consumer tech
- "Redefine the future of digital entertainment"
- €623M revenue, profitable, diversified
