# 平台分发与增长

> 来源: 行业报告、案例研究、Telegram 官方文档

## 1. Mini App 曝光渠道

### Bot 内启动
- **Menu Button**: Bot 聊天界面左下角菜单按钮，一键打开 Mini App
- **Inline Button**: 消息附带的按钮，引导用户打开
- **Bot 消息推送**: 主动发送消息（需用户先 /start）
- **Keyboard Button**: 自定义键盘按钮

### 直接链接
- `t.me/botname/appname`: 标准深度链接
- `t.me/botname/appname?startapp=xxx`: 带参数的深度链接（用于追踪/邀请）
- 支持 `mode=compact` 以小窗口模式打开

### 内联模式 (Inline Mode)
- 用户在任意聊天输入 `@botname` 触发
- 返回游戏卡片/邀请卡片供用户分享
- 零成本触达用户的社交网络

### Mini App 目录
- Telegram 官方 Mini App 商店/目录
- 按分类浏览和搜索
- 需满足质量标准才能收录

### Attachment Menu
- 出现在聊天附件菜单中
- 可在任何聊天类型中启动

## 2. 群组与频道运营

### 频道策略
- 创建官方频道发布更新、活动、排行榜
- 定期内容更新保持用户关注
- 频道内嵌 Mini App 链接按钮
- 置顶关键活动消息

### 群组策略
- 创建玩家社群促进交流
- Bot 自动发送排行榜/活动通知
- 群组专属奖励激励加入
- 管理员工具（踢人/禁言/公告）

### 社区运营节奏
- 每日: 签到提醒、每日任务
- 每周: 周榜公布、周末活动
- 每月: 赛季结算、新版本预告
- 里程碑: 用户数突破庆祝、特别活动

## 3. Telegram Ads

- 官方广告平台，投放到频道消息流
- 支持按 CPM 计费
- 可用 Stars 收入直接投放
- 精准定向: 频道主题/语言/地区
- 创意要求简洁，配合 Bot 链接

## 4. 病毒传播机制

### 邀请奖励系统
- 邀请人和被邀请人双向奖励
- 多级邀请奖励（1级直邀 + 2级间接）
- 邀请排行榜 + 额外奖励
- 研究表明邀请系统可提升获客 16%，LTV 提升 25%

### Squad/公会系统
- 组队机制（如 Notcoin 的 Squad）
- 团队排名竞争
- 公会专属福利
- 促进社交绑定和留存

### 社交竞技
- 好友排行榜（25% 留存提升）
- PVP 对战/挑战
- 成就/徽章分享
- #Hashtag 传播（Hamster Kombat 数百万社交曝光）

### 任务系统
- 关注频道任务
- 邀请好友任务
- 加入群组任务
- 分享到社交媒体任务

### Story 分享
- shareToStory() API 直接分享到 Telegram Stories
- 自定义分享图片/视频 + 链接
- 24小时可见，触达关注者

## 5. Bot 消息策略

### 主动触达（需用户已启动 Bot）
- 活动开始/结束提醒
- 每日签到提醒
- 好友超越通知
- 限时活动/促销推送

### 消息频率控制
- 避免过度推送导致用户屏蔽
- 关键通知 vs 营销消息区分
- 提供通知偏好设置
- 遵循 Telegram 反垃圾消息政策

### 召回策略
- 7天/14天/30天 不活跃用户分层召回
- 个性化召回内容（你的XXX等你领取）
- 回归奖励
- A/B 测试召回文案效果

## 6. 增长飞轮模型

```
新用户加入 → 体验核心玩法 → 产生社交行为
    ↑                              ↓
  邀请奖励 ← 分享给好友 ← 获得奖励/成就
```

### 关键指标
- DAU/MAU 比率（健康值 > 20%）
- K-factor（病毒系数，目标 > 1）
- 次日留存 / 7日留存 / 30日留存
- ARPU / ARPPU
- 邀请转化率

## 7. 成功案例增长数据

| 游戏 | 峰值用户 | 核心增长策略 |
|------|---------|-------------|
| Notcoin | 3000万+ | 邀请奖励 + Squad + 代币预期 |
| Hamster Kombat | 3亿+ | 社交媒体病毒传播 + 每日签到 |
| Catizen | 2000万+ | TON 生态整合 + NFT + 合成玩法 |
| Pixelverse | 1500万+ | 点击挖矿 + PvP 对战 |

## 8. 平台合规注意事项

- 不得发送垃圾消息
- 支付功能需遵守 Telegram ToS
- 数字商品必须用 Stars 支付（iOS/Android）
- 用户数据保护（GDPR 等）
- 游戏内容合规（无赌博/色情/暴力）
